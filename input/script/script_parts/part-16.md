# 16강 · Pro Virtual Try-On · 옷가게 가상 모델 입히기

> **결과물**: 인스타 릴스 (9:16 세로) · 옷 한 벌을 다양한 가상 모델 5컷에 입힌 정지샷 시리즈

[META]
이번 파트는 힉스필드 풀코스 16강. 30분 동안 진행해요. 핵심 비유는 "종이 인형 옷 갈아입히기"(Pro Virtual Try-On). 가상 클라이언트는 B 옷가게 셀러님이에요. 6강 Soul Reference·9강 Kling·13강 Cinema Studio에서 만난 그 페르소나 다시 돌아와요. Marketing Studio 9가지 모드 중 8번째 모드를 만나는 회차예요. 시작합니다.

[OVERVIEW]
오늘 만들 것 3가지예요. 첫째, Pro Virtual Try-On(프로 가상 착용 · 옷·뷰티 가상)이 왜 옷가게 셀러에게 마법 같은 기능인지 짚을 거예요. 둘째, 옷 사진 1장 + 모델 사진 1장 = 입은 모습 즉시 생성 흐름을 따라가볼 거예요. 셋째, B 옷가게 셀러 페르소나로 옷 한 벌을 다양한 가상 모델 5컷에 입혀서 인스타 릴스 정지샷 시리즈 1세트 받을 거예요.

[HOOK]
B 옷가게 셀러님이 평소 인스타 운영하면서 이런 고민 하셨을 거예요.
"옷 사진은 잘 찍었는데 모델이 없어서 옷만 사진으로 보여주니 임팩트가 약해요."
"모델 한 분 부르면 하루 촬영비 30~50만원이고 옷마다 부르기 부담돼요."
"같은 옷을 여러 모델·여러 자세로 찍어서 시리즈 만들고 싶은데, 이게 가능해요?"
이 고민, 오늘 한 방에 풀려요.
힉스필드 Marketing Studio에 Pro Virtual Try-On(프로 가상 착용) 모드가 있어요.
"종이 인형 옷 갈아입히기" 비유가 정확해요. 어렸을 때 종이 인형에 옷 종이 갖다 대면 입은 모습 되었던 거 기억나시죠.
디지털로 그 마법이 가능해진 거예요. 옷 사진 1장 + 모델 사진 1장 = 입은 모습 즉시 생성.
모델은 본인 사진도 OK, AI 가상 모델도 OK예요. 가상 모델은 다양한 인종·체형·자세로 무한 변형 가능해요.
오늘 같은 옷 한 벌을 5컷 다른 모델로 변형해서 인스타 릴스 정지샷 시리즈 만들어봐요.

[CONCEPT 1]
Pro Virtual Try-On(프로 가상 착용 · 옷·뷰티 가상 입히기) 부터 짚을게요.
"가상 착용"은요. 글자 그대로 "가상으로 옷을 입어보는 것"이라는 뜻이에요. 실제 모델 부르지 않고 사진만으로 입은 모습을 보여주는 기술이에요.
온라인 쇼핑몰 가본 적 있으시죠. 요즘 일부 쇼핑몰에 "AR로 입어보기" 기능 있는 거 보셨을 거예요. 그게 바로 가상 착용 기술이에요.
힉스필드 Pro Virtual Try-On은요. 의류 셀러용으로 특화된 가상 착용 기능이에요.
종이 인형 옷 갈아입히기 비유가 정확해요. 종이 모델에 옷 종이 갖다 대면 입은 모습 즉시 보였잖아요. 그걸 디지털로 옮긴 거예요.
9가지 Marketing Studio 모드 중 8번째 모드예요. 14강 Product Review·15강 UGC와는 다른 카테고리(의류·뷰티 셀러용)예요.
B 옷가게 셀러님처럼 의류 판매하시는 분들이 가장 많이 쓰는 마법 같은 기능이에요.

[CONCEPT 2]
옷 사진 + 모델 사진 업로드 흐름을 짚을게요.
Pro Virtual Try-On 모드에 진입하면 두 가지 업로드 박스가 떠요.
첫째 박스, "Garment(가먼트 · 옷 사진)" 업로드.
둘째 박스, "Model(모델 사진)" 업로드.
옷 사진은요. 본인이 판매하는 옷을 평평하게 펼쳐 찍은 사진이 가장 좋아요. 또는 마네킹에 입혀진 사진도 OK예요. 배경이 깔끔할수록 결과 정확해요.
모델 사진은 두 가지 옵션이 있어요. 첫째, 본인이나 지인 사진 업로드(실제 모델 입은 모습 받기). 둘째, AI 가상 모델 선택(다양한 인종·체형·자세에서 골라 입히기).
B 옷가게 셀러님 같은 분은 가상 모델이 더 활용도 높아요. 모델 한 분이 못 보여주는 다양한 체형·인종·연령대를 한 번에 보여줄 수 있거든요.

[CONCEPT 3]
입은 모습 즉시 생성을 짚을게요.
업로드 마치고 Generate(만들기) 버튼 누르면 약 1~2분 안에 입은 모습이 나와요.
14강 Marketing Studio(5~10분)·15강 UGC(5~10분)보다 빨라요. Pro Virtual Try-On은 옷+모델 결합만 하기 때문에 가벼운 작업이에요.
크레딧 비용도 30~50크레딧으로 14강·15강(50~100)보다 절반 정도예요.
옷의 색상·질감·로고가 그대로 유지되는지 확인하세요. 가끔 옷이 살짝 변형되는 경우가 있어요. 그럴 땐 다시 생성하거나 옷 사진을 더 깔끔한 걸로 교체하면 돼요.
입은 모습이 어색하지 않은지(예: 옷이 모델 몸에 안 맞는 비율) 확인하세요. 어색하면 모델 사진을 다른 체형으로 바꿔서 재시도해요.

[CONCEPT 4]
다양한 모델로 5컷 변형을 짚을게요.
같은 옷 + 5명 다른 가상 모델 = 5컷 시리즈 만들 수 있어요.
각 모델은 인종·체형·자세 다 다르게 선택 가능해요.
첫 번째 컷, 20대 한국 여성 모델 + 자연스러운 자세.
두 번째 컷, 30대 한국 여성 모델 + 옆모습.
세 번째 컷, 20대 동남아 여성 모델 + 워킹 자세.
네 번째 컷, 30대 서양 여성 모델 + 정면 자세.
다섯 번째 컷, 40대 한국 여성 모델 + 자연스러운 미소.
이렇게 5컷 만들면 한 옷에 5가지 다른 분위기가 나와요. 인스타 릴스 정지샷 시리즈로 활용할 때 시청자가 다양한 체형에서 옷이 어떻게 보이는지 한 눈에 비교할 수 있어요.
같은 옷을 5컷 만들 때 크레딧은 약 150~250(컷당 30~50)이에요.

[CONCEPT 5]
인스타 릴스 정지샷 변환을 짚을게요.
Pro Virtual Try-On 결과는 정지 이미지(JPG·PNG)예요. 영상이 아니에요.
그래서 인스타 릴스로 활용하려면 정지샷 5컷을 영상 편집 앱에서 슬라이드쇼로 묶어야 해요. 6강 Soul Reference 정지샷을 릴스로 변환했던 거 기억나시죠.
간단한 방법은요. 인스타 앱 안에 "릴스" 메뉴에서 사진 5컷 선택 → 자동 슬라이드쇼 → 음악 추가. 이게 가장 쉬워요.
조금 더 정교하게 하려면 캡컷(CapCut)·인쇼트(InShot) 같은 무료 영상 편집 앱에서 슬라이드 전환 효과 추가하시면 돼요.
영상 길이는 15~20초가 적당해요. 너무 길면 시청 완료율 떨어져요.

왜 Marketing Studio? Pro Virtual Try-On은 Marketing Studio 안 9가지 모드 중 8번째 모드예요. 14강·15강과 진입 경로 같아요.

**2단계 · 9가지 모드 카드에서 Pro Virtual Try-On 클릭**
Marketing Studio 들어가면 9가지 광고 모드 카드가 보여요.
TV Spot · UGC · Tutorial · Product Review · Unboxing · UGC Virtual Try-On · Hyper Motion · **Pro Virtual Try-On** · Wild Card 순서로 있어요.
8번째에 있는 "Pro Virtual Try-On" 카드 클릭하세요.

왜 Pro Virtual Try-On? "프로 가상 착용"이라는 이름 그대로 의류 셀러용 전문 가상 입히기 모드예요. UGC Virtual Try-On(6번째)은 일반 사용자 톤, Pro Virtual Try-On(8번째)은 셀러 전문 톤이라 결과가 더 정교해요.

**3단계 · 옷 사진 업로드 (Garment 박스)**
Pro Virtual Try-On 작업 화면 들어가면 "Garment(옷)" 업로드 박스가 첫 번째로 떠요.
본인이 판매할 옷 사진을 업로드하세요. 평평하게 펼친 사진이나 마네킹 사진이 가장 좋아요.
연습용으로 옷 사진 한 벌 미리 준비해두세요. 본인 가게 옷 없으면 무료 이미지 사이트(Unsplash·Pexels)에서 옷 사진 다운로드해서 써보세요.

왜 깔끔한 옷 사진? 셰프(AI)가 옷의 색상·질감·로고를 정확히 인식해야 모델에 정확히 입혀져요. 배경이 복잡하거나 옷이 구겨져 있으면 결과가 부정확해질 수 있어요.

**4단계 · 가상 모델 선택 (Model 박스)**
"Model(모델)" 업로드 박스 또는 "Choose AI Model(AI 모델 선택)" 옵션이 떠요.
오늘은 가상 모델 옵션 사용해요. "AI Model Gallery" 또는 비슷한 메뉴 클릭하세요.
다양한 가상 모델 갤러리가 떠요. 인종·성별·연령대·체형 필터로 첫 번째 모델을 선택하세요.
예: "20대 한국 여성 + 자연스러운 자세" 모델 선택.

왜 AI 가상 모델? 본인이나 지인 사진보다 다양한 체형·연령대를 한 번에 보여줄 수 있어요. 옷이 어떤 체형에 어울리는지 시청자가 비교 가능해서 인스타 릴스 효과 높아요.

**5단계 · Generate(만들기) 버튼 클릭 + 결과 확인 (1번째 컷)**
"Generate" 버튼 클릭하세요.
크레딧 차감 메시지 미리 확인하고 누르세요. 약 30~50크레딧이에요.
입은 모습 만드는 데 1~2분 걸려요. 14강·15강(5~10분)보다 훨씬 빨라요.
완료되면 화면에 옷 입은 1번째 컷이 떠요. 옷의 색상·질감이 그대로인지, 모델에 자연스럽게 입혀졌는지 확인하세요.

왜 미리 확인? 30~50크레딧이라도 5컷 만들면 150~250크레딧이에요. 가볍지 않아요. 한 번 결과 보고 마음에 들면 5컷 진행, 마음에 안 들면 옷 사진·모델 다시 골라요.

**6단계 · 모델 변경하여 2번째 컷 생성**
1번째 결과 확인 후 같은 옷으로 다른 모델 입혀보세요.
"Model" 박스에서 다른 가상 모델 선택. 예: "30대 한국 여성 + 옆모습" 모델.
"Generate" 버튼 다시 클릭. 약 30~50크레딧 추가.

왜 다른 모델? 같은 옷이 체형·자세·연령대 따라 어떻게 다르게 보이는지 시청자에게 비교해주려고요. 한 모델만으로는 옷의 다양한 매력 못 보여줘요.

**7단계 · 3·4·5번째 컷 반복 (총 5컷 시리즈)**
6단계 반복해서 3·4·5번째 컷 만드세요.
3번째 컷, 20대 동남아 여성 + 워킹 자세.
4번째 컷, 30대 서양 여성 + 정면 자세.
5번째 컷, 40대 한국 여성 + 자연스러운 미소.
총 5컷 만드는 데 약 150~250크레딧 들어요.

왜 다양한 인종·연령대? B 옷가게 셀러님 인스타 팔로워가 한국 여성만이 아닐 수 있어요. 다양한 체형 보여주면 더 많은 잠재 고객이 "나도 이거 입으면 잘 어울리겠다" 생각해요.

**8단계 · 5컷 다 Download + Assets 저장**
5컷 결과 화면에서 각 컷 옆 "Download" 버튼 클릭하세요. 컴퓨터에 5개 파일 저장돼요.
"Save to Assets" 버튼도 클릭하세요. 힉스필드 클라우드에 백업돼요.

왜 다 저장? 인스타 릴스 만들 때 5컷 다 필요해요. 그리고 다음에 같은 옷 다른 모델 추가하고 싶을 때 Assets에서 옷 사진 다시 꺼내 쓸 수 있어요.

**9단계 · 폰으로 옮기기**
다운로드한 5컷을 카카오톡 본인한테 보내거나 구글 드라이브로 폰에 옮기세요.
나중에 인스타 앱에서 슬라이드쇼 만들 때 폰에 있어야 해요.

왜 폰으로? 인스타 앱은 폰 전용이에요. 컴퓨터에서 직접 인스타 릴스 업로드 안 돼요.

**10단계 · 인스타 앱 → 릴스 → 5컷 슬라이드쇼**
인스타 앱 → "+" 버튼 → "릴스" 선택 → 사진 5컷 모두 선택 → 자동 슬라이드쇼 생성.
음악 추가는 인스타 안에서 추천 음악 중 트렌딩 음악 한 곡 선택하세요.
캡션은 옷 정보 + 5가지 체형으로 보는 후기 형식으로 짧게 적어주세요.
영상 길이는 15~20초로 맞추세요(컷당 3~4초).

왜 인스타 안에서 슬라이드쇼? 가장 쉽고 빨라요. 캡컷·인쇼트 같은 외부 앱 안 써도 충분해요. 처음엔 가장 쉬운 방법으로 익히세요.

**11단계 · 게시 + 결과 확인**
"공유" 버튼 클릭해서 인스타 릴스 게시하세요.
게시 직후 본인 인스타 프로필에서 릴스 탭 가서 확인해보세요.
시청 완료율·좋아요 수·댓글 수 추적해보세요. 24시간 후 인사이트 확인이 가장 정확해요.

왜 24시간 후? 릴스는 첫 24시간 동안 알고리즘이 노출 결정해요. 그 데이터로 다음 옷 시리즈 만들 때 어떤 모델·자세가 인기였는지 학습 가능해요.
