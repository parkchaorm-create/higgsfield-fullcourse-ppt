# 05강 · Soul ID · 내 사진 학습으로 일관 캐릭터

> **회차 메타**: 30분 · 왕왕초보 · D 트레이너 (30대 남성 · 릴스·운동 시연 · 틱톡·릴스 위주)
> **결과물**: 인스타 피드 (1:1) · 일관 캐릭터로 운동 시연 인물 사진 1~2장
> **전제 조건**: 04강 완료 (Soul 2.0 첫 인물 사진 1장 받음)
> **T1 검증일**: 2026-05-02

[META]
이번 파트는 힉스필드 풀코스 5강. 30분 동안 진행해요. 핵심 비유는 "셰프 앞치마에 내 이름 새기기"(Soul ID). 가상 클라이언트는 D 트레이너님이에요. 내 사진 5장을 셰프에게 학습시켜서 매번 같은 얼굴로 운동 시연 인물 사진을 받는 방법을 익혀볼 거예요. 시작합니다.

[OVERVIEW]
오늘 만들 것 3가지예요. 첫째, Soul ID(솔 ID · 내 사진 학습으로 일관 캐릭터)가 왜 필요한지, 셰프 앞치마 비유로 이해할 거예요. 둘째, Character(캐릭터 · 인물 일관성) 메뉴에서 D 트레이너 사진 5장을 업로드해서 학습시킬 거예요. 셋째, 학습 완료된 캐릭터로 운동 시연 인물 사진 1~2장을 받을 거예요.

[HOOK]
D 트레이너님이 4강 끝나고 이런 생각하셨을 거예요.
"인물 사진 1장은 잘 됐는데... 이걸로 시리즈 콘텐츠 만들기는 좀 그래."
"매번 다른 얼굴이 나오는데 어떻게 같은 사람으로 시리즈 영상 만들지?"
"인스타 운동 시리즈에 매주 다른 얼굴이 나오면 좀 이상하잖아."
이거 진짜 흔한 고민이에요.
4강의 Soul 2.0은 매번 새 얼굴을 만들어요. 같은 프리셋 써도 얼굴은 매번 달라요.
오늘 그 한계를 깨는 마법 도구 가져왔어요.
"Soul ID(솔 ID · 내 사진 학습)"라는 기능이에요.
내 사진 5~10장을 셰프한테 보여주면, 셰프가 "아, 이 사람이구나" 하고 외워요.
그 다음부터는 매번 같은 얼굴로 새 사진 만들어줘요. 시리즈 콘텐츠 끝.

[CONCEPT 1]
Soul ID(솔 ID · 내 사진 학습으로 일관 캐릭터)부터 짚을게요.
Soul ID는요. 셰프 앞치마에 내 이름과 사진을 새기는 거예요.
앞치마라고 비유 한 이유가 있어요.
힉스필드 셰프(AI 모델)는 매번 새로운 사람을 만들어요. 손님 한 명 한 명 다른 거 만들 듯이요.
근데 Soul ID는 셰프 앞치마에 "이 사람을 만들어줘"라고 사진 5장을 새겨두는 거예요.
다음부터 그 셰프한테 주문하면, 셰프가 앞치마 보고 "아, 이 사람이구나" 하고 항상 같은 얼굴로 만들어요.
이게 일관 캐릭터(Consistent Character)의 마법이에요.

[CONCEPT 2]
왜 일관 캐릭터가 중요한지 짚을게요.
인스타·틱톡·유튜브에서 콘텐츠 잘 되는 채널들은 다 한 가지 공통점이 있어요.
"같은 사람이 매번 나와요." 채널 호스트가 일관되게 보여요.
구독자가 그 얼굴에 정 들고, 친근감 느끼고, 다시 와요.
근데 매번 다른 AI 얼굴 나오면? 구독자가 "이거 누구야?" 헷갈려요.
브랜드도 안 만들어져요.
D 트레이너님이 "운동 시리즈 영상 5편" 만들었는데 매번 다른 얼굴 나오면 시리즈 자체가 망가져요.
Soul ID 학습 한 번 하면 평생 같은 얼굴로 콘텐츠 100편이든 1,000편이든 만들 수 있어요.

[CONCEPT 3]
학습용 사진 가이드를 알려드릴게요.
Soul ID는 사진 5~10장 정도면 충분히 학습해요.
어떤 사진을 골라야 하나요? 3가지 원칙이에요.
첫째, 다양한 각도. 정면 1장·옆얼굴 1장·살짝 위에서 1장·살짝 아래에서 1장. 셰프가 얼굴 입체감 익히게요.
둘째, 다양한 표정. 미소 1장·진지한 얼굴 1장·운동하는 모습 1장. 표정마다 얼굴 근육이 달라져요.
셋째, 깨끗한 얼굴. 모자·선글라스·마스크 쓴 사진 X. 셰프가 얼굴을 못 봐요.
이 3가지만 지키면 학습 잘 돼요. 사진 화질은 폰 셀카 정도면 충분해요.

[CONCEPT 4]
학습 시간을 짚을게요.
Soul ID는 사진 업로드 후 5~10분 정도 학습 시간 걸려요.
셰프가 "음, 이 사람 얼굴 외우는 중…" 하는 시간이에요.
이 5~10분 동안 우리는 다른 일 해도 돼요. 화면 앞에 안 있어도 돼요.
학습 완료되면 알림(Notification · 화면 우측 상단 벨 아이콘) 와요.
완료 후엔 그 셰프한테 "운동 시연 사진 만들어줘" 하면, 매번 D 트레이너 본인 얼굴로 사진 나와요.
이 5~10분 한 번 투자가 평생 시리즈 콘텐츠 만들 수 있게 해줘요.

[CONCEPT 5]
주의 사항 1개 짚을게요.
Soul ID 학습은 본인 사진으로만 하세요. 친구·가족·연예인 사진 무단 학습 금지예요.
이건 초상권 문제예요.
SNS에 올린 다른 사람 사진 가져다가 Soul ID 학습시키면 법적 문제 될 수 있어요.
본인 얼굴은 자유. 본인 가족은 가족 동의 받고. 그 외는 금지.
강의에서는 항상 본인 얼굴 또는 본인 동의 받은 얼굴로만 시연해요. 안전하게.

[DEMO]
이제 D 트레이너 페르소나로 Soul ID 학습부터 첫 결과물까지 만들어볼게요. 본인 얼굴로 진행하면 더 좋아요.

**1단계 · 본인 사진 5장 준비**
폰 사진첩에서 본인 얼굴 사진 5장 골라서 컴퓨터로 옮겨주세요.
구체적으로 이런 5장 추천해요. 정면 1장·옆얼굴 1장·살짝 위에서 1장·미소 1장·진지한 얼굴 1장.
모자·선글라스 안 쓴 사진으로요.

왜 5장? 너무 적으면 셰프가 얼굴 못 외워요. 너무 많으면 학습 시간 길어져요. 5장이 균형점이에요.

**2단계 · 힉스필드 접속 + Character 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 메뉴 중 "Character"(8번째 메뉴)를 찾아서 클릭하세요.
Character는 한국어로 "캐릭터(인물 일관성)"예요.

왜 Character? Soul ID는 Character 메뉴 안에 있어요. Image 메뉴 아니에요.

**3단계 · Soul ID 또는 "Create New Character" 클릭**
Character 페이지에 들어가면 "Soul ID" 또는 "Create New Character"(새 캐릭터 만들기) 버튼이 보여요.
클릭하세요.
새 캐릭터 만들기 화면이 열려요.

왜 이 버튼? 셰프 앞치마에 내 이름 새기기 시작하는 시작점이에요.

**4단계 · 캐릭터 이름 입력**
입력 박스에 캐릭터 이름 입력하세요. 한국어로 그대로 입력해도 돼요.

```paste
D트레이너 (또는 본인이름_트레이너)
```

왜 이름? 나중에 셰프가 "어떤 캐릭터 만들어드릴까요?" 물어볼 때 이 이름으로 골라요. 본인 알아보기 쉬운 이름으로.

**5단계 · 사진 5장 업로드**
"Upload Photos" 또는 사진 업로드 박스가 보여요.
1단계에서 준비한 사진 5장을 드래그해서 박스 안에 끌어다 놓거나, "Browse" 클릭해서 선택하세요.
업로드 진행 바가 끝까지 차면 완료예요.

왜 5장 한 번에? 셰프가 다양한 각도·표정 한 번에 봐야 얼굴을 입체적으로 외워요.

**6단계 · "Train" 또는 "Create" 버튼 클릭**
사진 업로드 끝나면 화면 아래 "Train Soul ID" 또는 "Create Character" 버튼이 활성화돼요.
클릭하세요.
"학습 시작" 메시지가 떠요.

왜 이 버튼? 셰프가 진짜 앞치마에 내 사진 새기기 시작하는 순간이에요. 5~10분 기다리는 시간 시작이에요.

**7단계 · 학습 진행 화면 확인 + 5~10분 대기**
"Training in progress..." 같은 메시지가 떠요. 진행 바가 천천히 차요.
이 시간 동안 다른 일 하셔도 돼요. 화면 닫아도 학습은 백그라운드에서 계속 진행돼요.

왜 기다림? 셰프가 사진 5장 분석해서 본인 얼굴 입체 모델 만드는 중이에요. 한 번 학습하면 평생 써먹어요.

**8단계 · 학습 완료 알림 확인**
5~10분 지나면 화면 우측 상단 알림(Notification · 벨 아이콘)에 빨간 배지가 떠요.
클릭하면 "Your character is ready!" 메시지가 보여요.
또는 Character 페이지 다시 들어가면 본인 캐릭터 카드가 "Ready" 상태로 보여요.

왜 알림? 다른 일 하다가 알림으로 완료 시점 알 수 있어요. 화면 앞 계속 안 있어도 돼요.

**9단계 · Image 메뉴로 이동 + Soul 2.0 선택 + 캐릭터 적용**
이제 본인 얼굴로 사진 만들어볼게요.
화면 위쪽 "Image" 메뉴 클릭하세요.
"Soul 2.0" 선택하세요.
작업 화면에 들어가면 "Use Character" 또는 "Apply Soul ID" 옵션이 보여요. 클릭해서 8단계에서 만든 캐릭터(D트레이너)를 선택하세요.

왜 캐릭터 적용? 이 단계가 셰프한테 "오늘은 이 사람 얼굴로 만들어줘" 외치는 신호예요.

**10단계 · 운동 시연 프롬프트 입력 + 1:1 종횡비 + Generate**
캐릭터 적용된 상태에서 한국어로 프롬프트 입력하세요.

```prompt
헬스장에서 덤벨을 들고 운동하는 30대 남성 트레이너, 밝은 자연광, 정면 구도, 미소 짓는 모습
```

종횡비 "1:1"(인스타 피드용) 선택하고 "Generate" 클릭하세요.
20~40초 후 본인 얼굴로 운동 시연하는 사진이 떠요.

왜 운동 시연? D 트레이너 페르소나에 맞는 콘텐츠예요. 인스타 시리즈 첫 컷 만드는 거예요.

**11단계 · 같은 캐릭터로 1장 더 만들기 (시리즈 확인)**
프롬프트만 바꿔서 다시 Generate 하세요.

```prompt
헬스장 거울 앞에서 스트레칭하는 30대 남성 트레이너, 자연광, 측면 구도
```

새 사진이 떠요. 1번째 사진과 비교해보세요. 자세는 다른데 얼굴은 같아야 해요.
이게 일관 캐릭터의 마법이에요.

왜 두 번째? 시리즈 콘텐츠 가능성을 직접 눈으로 확인하려고요. "같은 얼굴 다른 자세" — 이게 진짜 시리즈 콘텐츠 시작점이에요.

[RECAP]
오늘 한 일 정리할게요.
첫째. Soul ID(솔 ID)는 셰프 앞치마에 내 이름과 사진을 새기는 기능이에요. 한 번 학습하면 매번 같은 얼굴로 사진 만들어줘요. 시리즈 콘텐츠의 핵심이에요.
둘째. D 트레이너 페르소나로 본인 사진 5장 업로드 → 5~10분 학습 → 본인 얼굴로 운동 시연 사진 1~2장 받았어요. 결과물 누적 2~3번 ✓.
셋째. 본인 사진만 학습 사용하세요. 친구·연예인 사진 무단 학습은 초상권 문제예요. 강의에서는 항상 본인 얼굴 또는 본인 동의 얼굴로만 진행해요.

[BRIDGE]
다음 시간엔 사진 1장 가지고 시리즈 만드는 또 다른 방법 익혀요.
B 옷가게 셀러 페르소나로 진행해요.
참고 사진 1장 셰프한테 보여주면서 "이 분위기로 5컷 더 찍어줘" 하는 거예요.
Soul Reference(솔 레퍼런스 · 이미지 → 이미지 변환)라는 기능이에요.
"옷가게 인스타 릴스 정지샷 시리즈 5컷" 한 번에 받는 방법, 다음 회차에서 같이 만들어봐요.
