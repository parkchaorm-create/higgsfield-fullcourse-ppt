이제 진짜 A 카페 사장 페르소나로 X 헤더 배너 1~2장 만들어볼게요.

**1단계 · 힉스필드 접속 + Image 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 두 번째 메뉴 "Image" 클릭하세요.

왜 Image? 오늘은 글자 들어간 배너(이미지) 만들거예요. Video 메뉴 아니에요.

**2단계 · GPT Image 2 카드 선택**
모델 카드들 중에서 "GPT Image 2" 카드를 찾으세요.
힉스필드 메인 화면에서도 4분할 피처 카드에 "GPT IMAGE 2" 가 있어요. 거기서도 들어갈 수 있어요.
클릭하세요.

왜 GPT Image 2? 글자 또렷하게 만드는 셰프예요. 다른 셰프(Soul·Nano Banana)는 글자 만들면 깨져요.

**3단계 · 한글 글자 프롬프트 입력 (X 헤더 배너)**
화면 가운데 텍스트 입력 박스에 한국어로 그대로 입력하세요.

```prompt
왼쪽 상단에 '코지 카페'라는 한글 손글씨 큰 글씨, 가운데에 '동네 단골 환영'이라는 한글 작은 글씨, 따뜻한 갈색 톤 배경, 자연광이 비치는 카페 인테리어, 미니멀 디자인, 가로 긴 배너 형식
```

왜 이 프롬프트? "한글 손글씨 큰 글씨"로 메인 글자 + "한글 작은 글씨"로 서브 문구 + "위치 지정"(왼쪽 상단·가운데) + "갈색 톤·자연광·카페 인테리어"로 분위기 + "가로 긴 배너 형식"으로 X 헤더용 모양까지 한 번에 지시했어요.

**4단계 · Aspect Ratio 16:9 선택**
종횡비 옵션에서 "16:9"를 클릭하세요.
가로 TV 액자 모양이에요. X 헤더에 가장 가까운 비율이에요.

왜 16:9? X(트위터) 헤더는 정확히 3:1인데 힉스필드 옵션에는 16:9가 가장 가까워요. X에 올린 후 미리보기에서 위치 조정하면 돼요.

**5단계 · Generate(만들기) 버튼 클릭**
"Generate" 클릭하세요.
20~40초 기다리면 글자 들어간 배너가 화면에 떠요.
글자 또렷하게 나왔는지 확인하세요. "코지 카페"·"동네 단골 환영" 한글이 깨지지 않았는지 보세요.

왜 글자 확인? GPT Image 2 셰프 신뢰도 직접 검증하는 순간이에요. 다른 셰프와 차이를 눈으로 봐요.

**6단계 · 마음에 안 들면 Regenerate**
글자가 살짝 깨지거나 위치가 마음에 안 들면 "Regenerate"(다시 만들기) 클릭하세요.
크레딧 약 1~3개 추가 차감돼요.
2~3번 시도해서 가장 또렷한 글자 1장 고르는 게 정상이에요.

왜 다시? AI 셰프도 글자 만들기는 어려운 일이라 한 번에 100% 완벽 안 나와요. 평균 2~3번 시도해서 베스트 1장 고르세요.

**7단계 · Download + Assets 저장**
마음에 드는 배너 위에 마우스 올려서 "Download" + "Save to Assets" 둘 다 클릭하세요.
컴퓨터에도 저장하고 힉스필드 보관함에도 저장돼요.

왜 두 곳? 컴퓨터 다운로드는 X에 올릴 때 필요. Assets 보관은 나중에 다른 회차에서 활용할 때 필요해요.

**8단계 · 다른 글자 변형 1장 더 (선택)**
시간 여유 있으면 다른 문구로 1장 더 만들어보세요.

```prompt
가운데 크게 'COZY CAFE'라는 영문 손글씨, 아래쪽에 'Since 2020'이라는 작은 영문, 따뜻한 갈색 톤 배경, 자연광이 비치는 카페 인테리어, 미니멀 디자인, 가로 긴 배너 형식
```

왜 영문 변형? 한글·영문 두 가지 다 또렷하게 만든다는 걸 직접 확인하는 거예요. 영문 가게 이름 쓰는 카페도 많으니까요.

**9단계 · X(트위터) 헤더로 업로드**
다운로드한 배너를 X 앱이나 웹에 들어가서 헤더로 올리세요.
프로필 → 프로필 편집 → 헤더 사진 변경 → 저장한 배너 선택 → 위치 살짝 조정 → 저장.

왜 X에? A 카페 사장님 같은 경우 X 활용도 낮을 수 있는데요. 같은 16:9 배너를 카카오톡 채널 배경, 페북 페이지 커버, 네이버 블로그 헤더에도 쓸 수 있어요. 한 번 만들고 여러 채널 활용이에요.
