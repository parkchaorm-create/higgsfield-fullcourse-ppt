# 06강 · Soul Reference · 사진 한 장으로 시리즈 만들기

> **회차 메타**: 30분 · 왕왕초보 · B 옷가게 셀러 (20대 · 인스타 위주 · 릴스)
> **결과물**: 인스타 릴스 정지샷 (9:16 세로) · 일관 분위기 옷 시리즈 5컷
> **전제 조건**: 05강 완료 (Soul ID 학습 경험)
> **T1 검증일**: 2026-05-02

[META]
이번 파트는 힉스필드 풀코스 6강. 30분 동안 진행해요. 핵심 비유는 "사진 한 장 보여주며 '이 분위기로'"(Soul Reference). 가상 클라이언트는 B 옷가게 셀러님이에요. 참고 사진 1장만 보여주면 셰프가 그 분위기 그대로 5컷 더 만들어주는 마법, 같이 익혀볼게요. 시작합니다.

[OVERVIEW]
오늘 만들 것 3가지예요. 첫째, Soul Reference(솔 레퍼런스 · 이미지 → 이미지 변환)가 5강 Soul ID와 어떻게 다른지 한 줄로 정리할 거예요. 둘째, B 옷가게 셀러 페르소나로 참고 사진 1장 업로드해서 분위기 복사하는 방법을 익힐 거예요. 셋째, 9:16 세로 액자(인스타 릴스용)로 일관 분위기 옷 시리즈 5컷을 받을 거예요.

[HOOK]
B 옷가게 셀러님이 5강 끝나고 이런 생각 하셨을 수 있어요.
"내 얼굴 학습은 좀 부담스럽고, 다른 방법은 없나요?"
"인스타 릴스에 옷 시리즈 5컷 한 번에 올리고 싶은데 매번 분위기가 다르게 나와요."
"참고로 보여주고 싶은 사진은 있는데, 글로 설명하기 어려워요."
이 고민들 오늘 한 방에 풀려요.
힉스필드에 Soul Reference(솔 레퍼런스)라는 기능이 있어요.
참고 사진 1장 셰프한테 보여주면서 "이 분위기로 5컷 더 만들어줘" 하면 끝이에요.
글로 설명할 필요 없어요. 사진 1장이 1,000 단어보다 정확해요.
얼굴 학습도 안 하고요. 그냥 분위기 복사예요.

[CONCEPT 1]
Soul Reference(솔 레퍼런스 · 이미지 → 이미지 변환)부터 짚을게요.
Soul Reference는요. 셰프한테 사진 1장 보여주면서 "이 분위기로 비슷하게" 부탁하는 기능이에요.
친구한테 식당 추천받을 때 비슷해요. "어떤 분위기 식당이야?" 물어보면 친구가 사진 1장 보여주잖아요. "이런 분위기" 한 마디면 끝.
Soul Reference도 똑같아요. 우리가 셰프한테 사진 1장 보여주면 셰프가 "아, 이 분위기 원하시는 거군요" 하고 그대로 만들어줘요.
글로 "따뜻한 갈색 톤, 자연광, 미니멀 인테리어, 옷걸이에 걸린 옷..." 이렇게 길게 설명할 필요가 없어요.
사진 1장이 1,000 단어보다 정확해요.

[CONCEPT 2]
5강 Soul ID와 6강 Soul Reference의 차이를 명확히 짚을게요.
Soul ID는 "내 얼굴" 학습이에요. 셰프 앞치마에 내 사진 5장 새겨두고, 매번 같은 얼굴로 만들어요.
Soul Reference는 "분위기 복사"예요. 사진 1장 보여주면서 "이 분위기로" 부탁하는 거예요. 얼굴 학습 안 해요.
B 옷가게 셀러님 같은 경우는요. 내 얼굴 보여주는 채널 아니잖아요. 옷 자체가 주인공이에요.
그래서 Soul ID보다 Soul Reference가 딱 맞아요.
"이 분위기 옷걸이 사진처럼" 또는 "이 모델 자세처럼" — 분위기 복사로 시리즈 만들기 좋아요.
얼굴 학습 5~10분 기다릴 필요도 없고요.

[CONCEPT 3]
참고 사진(Reference · 참고 이미지 첨부) 잘 고르는 법을 알려드릴게요.
참고 사진 1장이 결과 80% 결정해요. 잘 고르는 게 핵심이에요.
3가지 원칙이에요.
첫째, 깨끗한 분위기. 잡다한 배경 없는 사진. 옷이나 모델이 주인공인 사진.
둘째, 비슷한 채널 분위기. 본인 인스타 분위기랑 닮은 사진. 갑자기 너무 화려한 사진 가져오면 셰프가 헷갈려요.
셋째, 핀터레스트(Pinterest · 이미지 검색 사이트)나 인스타에서 1장 골라오기. 본인 마음에 진짜 쏙 드는 1장이면 결과도 만족스러워요.
"옷걸이 미니멀 인테리어"·"자연광 편집숍 분위기" 이런 키워드로 검색하면 좋은 참고 사진 많이 나와요.

[CONCEPT 4]
9:16 세로 액자(인스타 릴스용)를 짚을게요.
4강에서 1:1 정사각 액자(인스타 피드용) 한 번 다뤘어요.
오늘은 9:16 세로 액자예요. 핸드폰 화면 통째로 차지하는 모양이요.
인스타 릴스·틱톡·유튜브 쇼츠가 다 9:16이에요.
B 옷가게 셀러님은 인스타 릴스가 메인 채널이라고 했죠. 그래서 9:16이 맞아요.
1:1로 만들면 릴스에 올릴 때 위아래 잘려요. 처음부터 9:16으로 만들어야 안전해요.
종횡비(Aspect Ratio) 선택만 1:1에서 9:16으로 바꾸면 끝이에요. 다른 건 똑같아요.

[CONCEPT 5]
Soul Location(솔 로케이션 · 시리즈용 배경)을 살짝 짚어둘게요.
오늘 핵심은 Soul Reference인데요. 함께 자주 쓰는 Soul Location도 알아두면 좋아요.
Soul Location은요. 시리즈 5컷 다 같은 배경에서 찍는 것처럼 만들어주는 기능이에요.
"카페 인테리어"·"옷가게 매대" 같은 배경 1개 정해두면 5컷 다 그 배경에서 나와요.
B 옷가게 셀러님이 매장 분위기 일관되게 보여주고 싶을 때 쓰면 좋아요.
오늘은 Soul Reference만 깊게 하고요. Soul Location은 다음 시리즈 작업 때 한 번 더 시도해보세요.

[DEMO]
이제 진짜 B 옷가게 셀러 페르소나로 9:16 시리즈 5컷 만들어볼게요.

**1단계 · 참고 사진 1장 준비**
핀터레스트(pinterest.com)나 인스타에서 본인 옷가게 분위기랑 비슷한 사진 1장 골라서 컴퓨터에 저장하세요.
구체적으로 이런 사진 추천해요. "옷걸이에 옷 걸린 미니멀 매장" 또는 "자연광 편집숍에서 모델이 옷 입고 있는 모습" 같은 거요.
검색 키워드: "minimal clothing store", "korean fashion editorial", "natural light boutique" 같은 영어 키워드도 좋아요.

왜 한 장만? 첫 시도는 1장으로 충분해요. 여러 장 한꺼번에 올리면 셰프가 어떤 분위기 따라야 할지 헷갈려요.

**2단계 · 힉스필드 접속 + Image 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 두 번째 메뉴 "Image" 클릭하세요.

왜 Image? 오늘은 시리즈 5컷도 결국 사진(이미지) 만드는 거예요. Video 메뉴 아니에요.

**3단계 · Soul 2.0 카드 선택**
모델 카드들 중에서 "Soul 2.0" 또는 "Soul" 카드를 클릭하세요.
4강에서 한 번 들어갔던 곳이라 익숙할 거예요.

왜 Soul 2.0? Soul Reference 기능은 Soul 2.0 안에 있어요. 다른 셰프(Nano Banana, GPT Image 2)는 이 기능 없어요.

**4단계 · "Reference" 또는 "Soul Reference" 옵션 활성화**
Soul 2.0 작업 화면에서 "Reference"·"Soul Reference"·"Image Reference" 같은 이름의 옵션이 보여요.
보통 화면 한 쪽에 사진 업로드 박스가 같이 있어요. 그 옵션을 켜거나 클릭하세요.

왜 이 옵션? 셰프한테 "글로 설명 말고 사진 보여줄게요" 신호 주는 거예요.

**5단계 · 1단계 참고 사진 업로드**
업로드 박스에 1단계에서 준비한 참고 사진 드래그해서 끌어다 놓으세요.
업로드 진행 바가 끝까지 차면 미리보기 사진이 박스 안에 떠요.

왜 드래그? 가장 빠른 업로드 방법이에요. "Browse" 버튼 클릭해서 선택해도 같은 결과예요.

**6단계 · 한국어 프롬프트 입력**
화면 가운데 텍스트 입력 박스에 한국어로 그대로 입력하세요.

```prompt
편집숍 매대 앞에서 옷을 입고 있는 20대 한국 여성 모델, 자연광, 미니멀 분위기, 정면 구도
```

왜 이 프롬프트? "편집숍 매대"로 배경 + "20대 한국 여성"으로 모델 + "미니멀 분위기"로 톤 통일 + "정면 구도"로 인스타 릴스에 잘 어울리는 구도까지 한 번에 지시했어요.

**7단계 · Aspect Ratio 9:16 선택**
입력 박스 아래 또는 옆에 종횡비 옵션이 있어요.
"9:16"을 클릭하세요. 4강에서는 1:1 골랐는데 오늘은 9:16이에요.
세로 핸드폰 화면 모양이에요.

왜 9:16? 인스타 릴스·틱톡·유튜브 쇼츠는 모두 9:16 표준이에요. 1:1로 만들면 릴스에 올릴 때 위아래 잘려요.

**8단계 · Generate(만들기) 버튼 클릭 + 첫 1컷 받기**
"Generate" 버튼 클릭하세요. 셰프한테 "주문 시작!" 외치는 신호예요.
20~40초 기다리면 첫 사진이 화면에 떠요.
참고 사진과 분위기 비슷한지 확인하세요.

왜 첫 컷부터 확인? 분위기가 너무 다르게 나오면 참고 사진을 바꾸는 게 빠른 답이에요. 5컷 다 만들고 나서 후회하면 늦어요.

**9단계 · 프롬프트만 살짝 바꿔서 2컷~5컷 받기**
첫 컷 분위기 마음에 들면, 프롬프트 살짝씩 바꿔서 4번 더 Generate 하세요.
예시로 이렇게 변형할 수 있어요.

```prompt
편집숍 매대 앞에서 옷을 입고 있는 20대 한국 여성 모델, 자연광, 미니멀 분위기, 측면 구도
```

```prompt
옷걸이 옆에서 옷을 고르고 있는 20대 한국 여성 모델, 자연광, 미니멀 분위기, 정면 구도
```

```prompt
거울 앞에서 옷을 입어보는 20대 한국 여성 모델, 자연광, 미니멀 분위기, 정면 구도
```

```prompt
매장 입구에서 옷가방을 들고 미소 짓는 20대 한국 여성 모델, 자연광, 미니멀 분위기, 정면 구도
```

왜 프롬프트만 변경? 참고 사진은 그대로 두고 프롬프트만 살짝씩 바꾸면 분위기는 통일된 채로 5컷 다른 자세·구도 받을 수 있어요. 시리즈 콘텐츠 핵심이에요.

**10단계 · 5컷 모두 Assets로 저장**
5컷 마음에 드는 거 다 Download 또는 Save to Assets 버튼 눌러서 저장하세요.
Assets에 저장해두면 나중에 다른 회차에서 영상 만들 때 가져다 쓸 수 있어요.

왜 모두 저장? 시리즈 5컷은 같이 인스타 캐러셀(여러 장 게시물)이나 릴스 정지샷 시퀀스로 한 번에 올릴 수 있어요. 묶어서 보관하세요.

**11단계 · 폰으로 옮겨서 인스타 릴스 정지샷으로 활용**
저장한 5컷을 카카오톡 본인한테 보내거나 구글 드라이브로 폰에 옮기세요.
인스타 앱 → 새 게시물 → 릴스 또는 캐러셀 선택 → 5컷 순서대로 올리기.

왜 정지샷? 영상 안 만들고도 사진 5컷을 릴스에 슬라이드로 보여줄 수 있어요. 옷가게 시리즈 콘텐츠 가장 가벼운 형태예요.
