## 5체크 (필수 · 1번 강의 형식 승계)

### 1. 시간 표기
- [x] **포함하지 않음** (관악강의·1번 강의 패턴 일치)
- 회차 길이는 강사용 큐시트(`강사매뉴얼/`)에 분단위로 기록하되, 슬라이드·실습 본문에는 시간 표기 금지
- 예외: 실습 튜토리얼 헤더 박스의 "**소요 시간**: 25~35분" 한 줄만 허용 (강사가 영상 녹화 시 자율)

### 2. 이모지 사용 범위
- [x] bullet-text 시작 1개 (의무) — `🎬 Generate Video · 영상 만들기 첫 시작`
- [x] callout/quote 첫 글자 (의무) — ✅ ❌ 📺 💡 ⚠️ 📌 🐛 🚀 📋 🔗
- [ ] section-title 이모지 금지 (텍스트만)
- [ ] **회차당 이모지 8개 초과 금지** (왕왕초보 시각 산만 방지)
- [ ] bullet-text 첫 글자 외 이모지 금지

### 3. 톤·종결어미
- [x] **해요체** (`-요`, `-죠`, `-거든요`)
- 예: "여기를 누르면 새 화면이 나와요" ✓
- 예: "여기를 누르면 새 화면이 나옵니다" ✗
- 강사 노트의 노래방 본문(`.note-script-text`)은 강사가 직접 읽는 구어체 그대로 (한 문단 연속)
- **자기지칭 금지** (`저는`·`제가`) — 원본 스크립트에 있을 때만 유지

### 4. 숫자 표기
- [x] **아라비아** 숫자 (`30분`, `3가지`, `20회차`, `10크레딧`)
- 예외: "한 단계", "두 번째" 같은 일반어는 한글 허용
- 가격 표기는 `$34~39` 또는 `약 5만원` (강사 자율)

### 5. 외래어 표기 (왕왕초보 강화)
- [x] 영문 그대로 + **첫 등장 시 한글 풀이 1회** (슬라이드 단위 · `inline-glossary.md` v2)
  - 예: `Marketing Studio(마케팅 스튜디오 · 광고 자동 생성 메뉴)`
  - 예: `Soul 2.0(솔 2.0 · 인물 사진 생성 모델)`
  - 예: `Hermes Agent(헤르메스 에이전트 · UGC 트렌드 자동 분석)`
- bullet-text는 길이 제약상 풀이 생략 가능, bullet-detail에서 풀이
- **SVG 안에서는 영어 단독 노출 금지** (한글 병기 또는 한글만)
- **인스타·틱톡·유튜브·페북·X**는 한글 그대로 OK (이미 한국 표준어)

## 왕왕초보 추가 정책 (audience.tone_principles 6항 박제)

### 가-1. 비유 우선
- 모든 개념은 **일상 비유로 먼저 설명** 후 도구 용어 도입
- 비유 사전: `_design/dictionaries/비유사전.md` 30개 박제 — 임의 발명 금지
- 예: "Higgsfield = 셰프가 여러 명 있는 큰 주방" (회차 1 박제 후 회차 20까지 일관)

### 가-2. 클릭 가이드 (한 단계 한 동작)
- 실습은 `여기 누르세요 → 이 화면 나옴 → 다음 누르세요` 단계별 분해
- **묶음 금지**: "이거 누르고 저거 누르세요" → 두 단계로 분리
- 각 단계마다 "**왜 이 버튼?** 1줄 이유" 동반 (audience.tone_principles 6항)

### 가-3. 메뉴풀이 의무
- 모든 영어 메뉴명·전문 용어 **첫 등장 시 한글 풀이 괄호 병기** (슬라이드 단위)
- 사전: `_design/dictionaries/메뉴풀이사전.md` 50개 박제 — 임의 추측 금지
- 인터페이스 SSOT: `starter/input/materials/higgsfield-reference/higgsfield-interface-2026-05-02.md` (절대 우위)

### 가-4. 스크린샷 적극 활용 (1번 강의 v2 정책 예외)
- 1번 강의는 텍스트 위주(터미널) → 스크린샷 0건 정책 (`tutorial-screenshot-requirement.md` v2)
- 본 강의는 GUI 도구 → **스크린샷 적극 사용 예외 적용** 박제
- 정책 상세: `_design/screenshot-policy.md` 별도 SSOT
- 단 PPT 자체에는 스크린샷 임베드 금지 (튜토리얼 마크다운에서만)

### 가-5. 한 번에 한 동작
- bullet-text 1개 = 동작 1개 (또는 개념 1개)
- 한 슬라이드 bullet-text는 max 2 (`density-budget.json`)
- "그리고", "이어서" 같은 묶음 어미 사용 자제

### 가-6. 왜 하는지 설명
- 단계마다 "왜?" 1줄 동반 — 수강생 동기 유지
- bullet-detail 첫 문장이 "왜" → 둘째 문장이 "어떻게" 권장

## 결과 약속·과장 표현 금지 (공공 강의 톤 R 시리즈)
