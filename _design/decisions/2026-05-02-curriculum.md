## R1. ACT 구조 결정

**김마케팅**: 옵션 B(5 ACT)에 +1. 결과물 누적이 채널별로 명확. Marketing Studio를 별도 ACT로 띄우면 핵심 가치(Hermes Agent·UGC) 부각.
**이영상**: 옵션 B 동의. 다만 "마무리 3"은 너무 많음 → 2회면 충분. 기초 3 → 영상 6으로 조정 제안.
**박왕초**: 옵션 A(4 ACT)는 직관적이지만 "기초 5"가 너무 김. **첫 3회는 무조건 손에 잡혀야** 포기 안 함. 옵션 B의 "소개 3" 좋음.
**최교육**: 옵션 C(3 ACT)는 인지 단위 너무 큼 — 왕왕초보는 6회 묶이면 길 잃음. 옵션 B 지지. 단 ACT 간 휴식 슬롯(중간점검) 권장.
**정기술**: 옵션 B + Marketing Studio ACT 신설 핵심. Hermes Agent는 2026-04-23 신규라 강의 차별화 가치 ↑.

**합의**: **옵션 B (5 ACT)** 만장일치.
**최종**: ACT1 입문 3 / ACT2 이미지 5 / ACT3 영상 5 / ACT4 Marketing Studio 4 / ACT5 실전·마무리 3 = 20회.
**근거**: 왕왕초보 인지 단위(3~5회)와 Marketing Studio의 차별화 가치를 동시에 만족.

## R2. 결과물 채널 배정

**김마케팅**: 인스타 릴스가 가장 활용도 높음 → 최소 3회 배정. 페북 광고는 Marketing Studio ACT에서 묶기.
**이영상**: 유튜브 쇼츠는 시네마틱 모델(Sora 2·Cinema Studio 3.0) 한 회씩. 틱톡은 토킹헤드(Seedance 2.0) 1회면 충분.
**박왕초**: X(트위터)는 한국 카페 사장이 거의 안 씀 → 1회만 가볍게. 페북 일반 게시물 vs 광고 분리 필요.
**최교육**: 채널별 회차 배정 시 "결과물 1개 = 1회"가 인지부하 최소. 한 회에 2채널 동시 출시 금지.
**정기술**: 종횡비 매핑(9:16·1:1·4:5·16:9)을 회차마다 명시 → 학생이 헷갈리지 않음.

**합의**: 채널별 회차 분포: **인스타 릴스 4회 · 인스타 피드 2회 · 틱톡 2회 · 유튜브 쇼츠 3회 · 페북 광고 3회 · 페북 일반 1회 · X 1회 · 통합/혼합 4회**.
**최종**: 모든 5종 채널을 최소 1회+ 다루며 인스타 릴스 가중. 한 회 = 한 채널 원칙.

## R3. Higgsfield 기능 12~16개 선정

**정기술**: 16개 후보 중 Canvas(워크플로우)는 왕왕초보에 너무 추상 → 제외. 100+ Apps는 ACT5 통합 회차에서 1슬롯만.
**김마케팅**: Marketing Studio 4종(제품광고·UGC·Try-On·Hermes)은 모두 필수. 차별화 핵심.
**이영상**: 영상 모델 4종(Kling·Seedance·Sora·Veo)은 채널별 1:1 매핑. Cinema Studio 3.0은 광고용 1회.
**박왕초**: "크레딧 절약" 회차는 따로 빼지 말고 매 회차 끝에 **5분 크레딧 가계부** 코너로 분산 권장.
**최교육**: "결과물 채널 매칭"은 마지막 회차(20강) 종합 정리에 적합. Hermes Agent는 마케팅 ACT에 자연스럽게 녹임.

**합의**: **14개 핵심 기능 채택** + 매 회차 끝에 크레딧 코너 분산. Canvas·MCP는 강의 본편 제외(보너스 자료 후일).
**최종 14개**: 회원가입+체험 / Soul 2.0 / Soul ID / Soul Reference / Nano Banana Pro / GPT Image 2 / Kling 3.0 / Seedance 2.0 / Sora 2 / Veo 3.1 / Cinema Studio 3.0 / Marketing Studio 제품광고 / Marketing Studio UGC+Hermes / Pro Virtual Try-On.

## R4. 가상 클라이언트 4종 그대로 vs 수정

**박왕초**: A(카페)·C(식당)이 둘 다 음식이라 약간 겹침. 다만 운영 채널·연령 다르니 OK.
**김마케팅**: 4종 모두 마케팅 다양성 충분(F&B 2 · 패션 1 · 운동 1). 추가 없음.
**이영상**: D(트레이너) 운동 시연은 Seedance 립싱크와 궁합 좋음. 유지.
**최교육**: 회차마다 4종 다 다루지 말고 **회차당 1~2종만 깊게**. 인지부하 ↓.
**정기술**: 다양성 OK. 단 가상 클라이언트는 회차별 핵심 1종 + 보조 1종 구조 권장.

**합의**: **4종 그대로 유지**. 회차당 핵심 1 + 보조 1 (또는 단독) 적용.
**최종**: A(카페·40대) · B(옷가게·20대) · C(식당·50대) · D(트레이너·30대) 박제.
