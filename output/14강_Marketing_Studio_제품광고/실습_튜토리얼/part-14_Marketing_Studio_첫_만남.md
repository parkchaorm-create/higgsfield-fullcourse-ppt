## 1️⃣ Step 1 · 시연 1

이제 진짜 C 식당 메뉴 URL로 30초 페북 광고 영상 1편 만들어볼게요.

**1단계 · 힉스필드 접속 + Marketing Studio 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 메뉴 중 "Marketing Studio" 클릭하세요. "New" 배지 붙은 메뉴예요. 글로벌 네비 9번째예요.
또는 메인 화면 7개 액션 카드 중 "Marketing Studio" 카드 클릭해서 들어갈 수 있어요.

왜 Marketing Studio? 9~13강의 Video 메뉴와 다른 별도 메뉴예요. 광고 자동화가 핵심 차이라 메뉴가 따로 있어요.

**2단계 · 작업 화면 둘러보기**
Marketing Studio 작업 화면 들어가면 9~13강의 일반 영상 작업 화면과 좀 달라요.
보통 위쪽에 "Enter product URL or describe" 같은 URL 입력 박스
중간에 9가지 광고 모드 카드들
아래쪽에 추가 옵션들
이런 구조예요.

왜 둘러보기? URL 입력 박스 위치, 모드 카드 위치를 먼저 파악하면 다음 단계가 쉬워요.

**3단계 · 광고 모드 선택 (Product Review)**
9가지 모드 카드 중에서 "Product Review"(프로덕트 리뷰) 클릭하세요.
C 식당 메뉴 광고에 가장 무난한 모드예요. UGC도 좋지만 첫 시도는 Product Review가 안전해요.

왜 Product Review? "메뉴 후기" 분위기로 광고가 만들어져요. 음식 광고에 잘 맞는 톤이에요. UGC는 15강에서 깊이 다룰 거예요.

**4단계 · 제품 URL 입력 (또는 직접 입력)**
URL 입력 박스에 C 식당 메뉴 URL을 붙여넣으세요. 예를 들면 카카오톡 채널·네이버 블로그·네이버 플레이스 URL이에요.

```paste
https://place.naver.com/restaurant/123456789/home
```

(위 URL은 예시예요. 본인 식당 URL 또는 가상 클라이언트 URL을 사용하세요.)

URL이 없는 경우엔요. 직접 제품 정보를 한국어로 입력하셔도 돼요.

```prompt
한식당 "이모네 식당" 메뉴 광고. 대표 메뉴: 비빔밥(8천원), 불고기(1만2천원), 잡채(7천원). 위치: 서울 강남. 30년 전통, 가족 운영, 따뜻한 분위기. 페북 광고용.
```

왜 URL 또는 직접 입력? URL은 셰프가 자동으로 페이지 분석하지만, URL 없으면 직접 정보 입력해도 같은 결과 나와요. C 식당 사장님 본인 페이지 URL이 있다면 URL 입력이 더 편해요.

**5단계 · Aspect Ratio 4:5 + 길이 30초 선택**
종횡비 옵션에서 "4:5" 클릭하세요. 12강에서 익힌 페북 광고 표준이에요.
영상 길이 옵션에서 "30s"(30초) 클릭하세요. Marketing Studio는 30초가 표준이에요.

왜 4:5 + 30초? 페북·인스타 광고 시스템이 가장 좋아하는 형식이에요. 4:5 액자가 화면 많이 차지하고, 30초가 광고 한 편의 황금 길이예요.

**6단계 · Generate(만들기) 버튼 클릭**
"Generate" 또는 "Launch Campaign" 버튼 클릭하세요.
크레딧 차감 메시지 미리 확인하고 누르세요. 약 50~100크레딧이에요.
영상 만드는 데 5~10분 걸려요. 일반 영상보다 좀 더 걸려요.
화면에 진행 상황(URL 분석 → 콘셉트 추천 → 영상 생성 → 카피 생성) 단계별로 떠요.

왜 시간 더? 영상 + 카피 + 변형까지 한 번에 만들기 때문이에요. 단계 보이면서 어떻게 광고가 자동 생성되는지 학습 기회예요.

**7단계 · 광고 캠페인 결과 확인**
완료되면 화면에 결과 한 세트가 떠요.
첫째, 30초 광고 영상 (4:5)
둘째, 광고 카피 (광고 문구) 3가지 예시
셋째, 종횡비 변형 (9:16 인스타 릴스용·1:1 인스타 피드용)
영상 재생해서 자연스러운지, 카피 문구 마음에 드는지 확인하세요.
크레딧 잔액 확인. 약 50~100 줄었을 거예요.

왜 카피까지 확인? 광고는 영상만 좋아도 안 돼요. 문구가 시선 잡아야 영상 클릭해요. Marketing Studio가 만든 카피 3가지 중에서 본인 가게 톤에 맞는 것 1개 고르세요.

**8단계 · 마음에 안 들면 카피 또는 영상 부분 재생성**
영상이나 카피 마음에 안 들면 부분만 다시 만들 수 있어요.
"Regenerate Video"·"Regenerate Copy" 같은 버튼 따로 있을 거예요.
영상만 다시 만들면 약 30크레딧, 카피만 다시 만들면 약 5크레딧 정도예요.

왜 부분 재생성? 처음부터 다 다시 만들면 50~100크레딧 또 들어요. 부분만 다시 하면 절약돼요. 광고에서 영상은 OK인데 카피가 약하면 카피만 재생성하세요.

**9단계 · Download + Assets 저장 (3가지 종횡비 다)**
4:5(페북)·9:16(인스타 릴스)·1:1(인스타 피드) 3가지 다 따로 "Download" + "Save to Assets" 클릭하세요.
영상 파일 3개 + 카피 텍스트 1개가 컴퓨터에 저장돼요.

왜 다 저장? 같은 광고를 페북·인스타 릴스·인스타 피드 3채널에 동시 배포할 수 있어요. 한 번 만들고 3채널 활용이에요.

**10단계 · 페북 광고 매니저로 업로드 (광고 캠페인 시작)**
페북 광고 매니저(Facebook Ads Manager)에 접속하세요.
새 캠페인 → 광고 형식 "동영상" → 다운로드한 4:5 영상 업로드 → 광고 카피 붙여넣기 → 타겟·예산 설정 → 게시.
인스타 광고도 같이 운영하려면 인스타용 9:16·1:1 영상 따로 업로드 가능해요.

왜 광고 매니저? C 식당 사장님 진짜 광고 운영 시작점이에요. Marketing Studio 결과물이 진짜 광고 캠페인이 되는 순간이에요. 페북·인스타 동시 광고로 도달 범위 넓히세요.
