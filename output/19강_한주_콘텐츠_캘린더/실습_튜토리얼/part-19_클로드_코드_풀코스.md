## 1️⃣ Step 1 · 시연 1

이제 진짜 본인 페르소나로 한 주 7편 콘텐츠 캘린더 만들어볼게요.

**1단계 · 본인 페르소나 1종 선택**
A 카페·B 옷가게·C 식당·D 트레이너 중 본인 페르소나에 가장 가까운 1종 선택하세요.
선택한 페르소나로 한 주 콘텐츠 7편 만들 거예요.
연습용으로 본인 가게·콘텐츠 분야가 아니어도 페르소나 1개 선택해서 시도해보세요.

왜 페르소나 1종? 7편을 한 페르소나로 통일해야 일관된 채널 분위기 나와요. 7편마다 다른 페르소나면 채널 정체성 흐려져요.

**2단계 · 한 주 7편 주제 미리 정하기 (5분)**
요일별 테마 프레임워크 또는 시리즈 프레임워크 1개 선택해서 7편 주제 미리 정하세요.

예: D 트레이너 페르소나 + 요일별 테마.
1편(월) - 운동 시작 동기 부여 정지샷.
2편(화) - 정확한 자세 정보 카드.
3편(수) - 1주 비포·애프터 비교 영상.
4편(목) - "운동할 때 무릎 아파요" 질문 답변 영상.
5편(금) - 시네마틱 운동 시연 영상.
6편(토) - 운동 후 일상 정지샷 시리즈.
7편(일) - 다음 주 운동 챌린지 예고 광고.

종이나 메모 앱에 7편 주제 미리 적어두세요.

왜 미리? 힉스필드에 들어가서 즉흥적으로 정하면 시간 5배 더 걸려요. 미리 7편 주제 정해두고 한 번에 빠르게 만드는 게 효율적이에요.

**3단계 · 모델 분배 결정 (3분)**
2단계 7편 주제별로 어떤 모델·모드 사용할지 결정하세요.

예: D 트레이너 7편 모델 분배.
1편 - Soul 2.0 (인물 정지샷).
2편 - GPT Image 2 (글자 들어간 카드).
3편 - Kling 3.0 (9:16 비포·애프터 영상).
4편 - Seedance 2.0 (토킹헤드 질문 답변).
5편 - Cinema Studio (시네마틱 운동 시연).
6편 - Soul Reference (일상 정지샷 시리즈).
7편 - Marketing Studio TV Spot 또는 UGC (광고 톤 예고).

본인이 익숙한 모델 위주로 분배하세요. 모르는 모델은 4·5단계에서 시도하지 말고 익숙한 모델로 7편 채워도 OK예요.

왜 모델 분배? 한 모델로 7편 만들면 분위기 단조로워요. 다양한 모델 골고루 사용해야 채널 매력 느껴져요.

**4단계 · 힉스필드 접속 + 첫 번째 콘텐츠 생성 (1편 · 인물 정지샷)**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

"Image" 메뉴 → "Soul 2.0" 선택 → 한국어 프롬프트 입력 → 1:1 액자 → Generate.
4강에서 만들었던 흐름이에요. 익숙해서 빠르게 진행 가능.
약 5크레딧, 1~2분 소요.

```prompt
30대 남성 트레이너가 헬스장에서 자신감 있게 카메라를 보고 있는 인물 사진. 운동복 차림. 따뜻한 조명. 동기 부여 분위기. 1:1 정사각.
```

왜 첫 번째 인물 정지샷? 가벼운 작업으로 시작해서 흐름 잡아요. 갑자기 영상부터 시작하면 부담스러워요.

**5단계 · 두 번째 콘텐츠 생성 (2편 · 정보 카드)**
"Image" 메뉴 → "GPT Image 2" 선택 → 한국어 프롬프트 입력 → 1:1 액자 → Generate.
8강에서 만들었던 흐름이에요.
약 5크레딧, 1~2분 소요.

```prompt
한국어로 "스쿼트 정확한 자세 3가지" 큰 제목. 그 아래 작은 글자로 "발 어깨 너비, 무릎 발끝 방향, 허리 곧게". 미니멀 디자인. 1:1 정사각. 헬스장 분위기 배경.
```

왜 정보 카드? 인물 사진 다음에 정보·팁 콘텐츠로 가치 전달. 시청자가 본인 채널에 정보 찾으러 오게 하는 콘텐츠.

**6단계 · 세 번째 콘텐츠 생성 (3편 · 비포·애프터 영상)**
"Video" 메뉴 → "Kling 3.0" 선택 → 9:16 액자 → 한국어 프롬프트 → Generate.
9강에서 만들었던 흐름이에요.
약 6크레딧, 5~10분 소요.

```prompt
30대 남성이 1주 운동 시작 전과 1주 후 비포·애프터 비교 영상. 첫 5초는 시작 전 모습, 다음 5초는 1주 후 모습. 9:16 세로 영상.
```

왜 영상부터 본격 시작? 6편이 영상이라 영상 1편씩 완성해가야 진도 빨라요.

**7단계 · 네·다섯·여섯 번째 콘텐츠 빠른 진행 (4·5·6편)**
같은 흐름으로 4·5·6편 빠르게 진행하세요.
4편 - "Video" 메뉴 → "Seedance 2.0" → 토킹헤드 영상 → 9:16. 약 6크레딧.
5편 - "Video" 메뉴 → "Cinema Studio 3.5" → 시네마틱 → 9:16. 약 10~15크레딧.
6편 - "Image" 메뉴 → "Soul Reference" → 정지샷 시리즈 → 9:16. 약 5~10크레딧.

각 편 한국어 프롬프트는 2단계 미리 정해둔 주제 그대로 활용.

왜 빠른 진행? 흐름 끊기지 않고 연속으로 만들면 1~2시간 안에 완료. 중간에 다른 일 하면 흐름 잃어 시간 더 걸려요.

**8단계 · 일곱 번째 콘텐츠 생성 (7편 · 광고 톤 예고)**
"Marketing Studio" 메뉴 → 14강 또는 17강 모드 선택 → 한국어 프롬프트 → 1:1 또는 9:16 → Generate.
약 70~150크레딧, 5~10분 소요.

```prompt
"다음 주 1주 운동 챌린지 시작" 예고 광고. 30대 남성 트레이너가 카메라 보고 자신감 있게 안내. 시네마틱 톤. 길이 15초. 9:16 세로.
```

왜 7편 광고 톤? 한 주 마무리에 다음 주 예고로 시청자 기대감 만들어요.

**9단계 · 7편 다 Download + Assets 저장**
힉스필드 Assets 또는 각 결과 화면에서 7편 다 "Download" 버튼 클릭.
컴퓨터에 7개 파일 저장.
"Save to Assets" 버튼도 다 클릭.

왜 다 저장? 다음 주 콘텐츠 만들 때 이번 주 7편 일부 재활용 가능. 또는 다른 채널에 다른 시점에 다시 게시.

**10단계 · 폰으로 옮기기**
다운로드한 7편을 카카오톡 본인한테 보내거나 구글 드라이브로 폰에 옮기세요.

왜 폰으로? 인스타·유튜브 쇼츠 게시 예약은 폰 앱이 가장 편해요.

**11단계 · 채널별 게시 예약 (월~일)**
인스타 앱·유튜브 앱 활용해서 월~일 매일 1편씩 게시 예약하세요.

월요일 - 1편 인물 정지샷 인스타 피드 1:1.
화요일 - 2편 정보 카드 인스타 피드 1:1.
수요일 - 3편 비포·애프터 영상 인스타 릴스 9:16.
목요일 - 4편 토킹헤드 영상 인스타 릴스 9:16 + 유튜브 쇼츠.
금요일 - 5편 시네마틱 영상 인스타 릴스 9:16 + 유튜브 쇼츠.
토요일 - 6편 정지샷 시리즈 인스타 피드 1:1.
일요일 - 7편 예고 광고 인스타 릴스 9:16.

게시 예약은 인스타 비즈니스 계정이면 가능해요. 일반 계정이면 매일 직접 올려야 해요.

왜 매일? 한 번에 7편 다 올리면 알고리즘이 1편만 노출하고 나머지는 묻혀요. 매일 1편씩이 가장 좋아요.

**12단계 · 한 주 후 결과 정리**
한 주 게시 후 인스타 인사이트 확인하세요.
어떤 편이 좋아요·시청 완료율 높았는지 데이터 수집.
다음 주 캘린더 만들 때 잘 된 편의 모델·톤 더 활용하시면 돼요.

왜 데이터 수집? 본인 시청자가 어떤 콘텐츠 좋아하는지 학습. 막연한 추측 대신 데이터 기반으로 다음 주 콘텐츠 결정.

> ✅ **성공 신호**: 위 마지막 결과 화면이 그대로 나옴

> ❌ **실패 시**: 수업 채널에 캡처 + "19강 Step 1" 키워드로 질문
