# 20강 실습 · 결과물 채널 매칭 종합 + 다음 단계 (포트폴리오)

> **소요 시간**: 25~35분 (수업 시간 내 완결)
> **목표 산출물**: 결과물 채널 매칭 종합 + 다음 단계 (포트폴리오) 직접 따라하기
> **안전망**: 1개 단계 중 마지막 직전까지 도달하면 합격
> **선행 조건**: 이전 회차 완료 (19강 산출물)

## 🚀 Step 0 · 빠른 셋업 (3분)

1. 책상 위 음료수 한 잔 준비
2. 휴대폰 무음 모드
3. 다른 창 모두 닫기 (집중)
4. 터미널(또는 PowerShell) 열기
5. 본 회차 PPT 강사 노트 화면 함께 띄워두기

---

## 1️⃣ Step 1 · 시연 1

이제 졸업 작업 시작해볼게요. 마지막 데모예요.

**1단계 · 채널 × 모델 매트릭스 노션·메모에 캡처**
오늘 콘셉트 1번에서 본 채널 × 모델 매트릭스를 본인 노션 또는 메모 앱에 캡처하세요.
스크린샷 찍어서 저장하거나 표를 그대로 메모 앱에 옮겨 적으세요.

본인 SSOT(단일 진실 공급원)로 사용할 거예요. 다음 콘텐츠 만들 때마다 이 표 한 장 보고 1초 안에 모델·모드 결정.

왜 SSOT? 매번 어떤 모델 쓸지 헤매면 시간 낭비예요. SSOT 하나 두면 결정 빠르고 일관돼요.

**2단계 · 노션 무료 계정 만들기 (없으면)**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://www.notion.so
```

본인 이메일로 무료 계정 만드세요. 1분 안에 가입 완료.
이미 노션 계정 있으면 2단계는 패스.

왜 노션? 회사·크리에이터가 가장 많이 쓰는 정리 앱이에요. 무료 플랜으로도 충분해요.

**3단계 · 노션 새 페이지 만들기 + 표지 페이지 작성**
노션 좌측 사이드바에서 "+" 또는 "New Page" 클릭.
페이지 제목: "본인 이름 · 힉스필드 풀코스 졸업 포트폴리오".
본문에 한 줄 자기소개 + 본인 SNS 채널 링크 적어주세요.

```paste
이름: 홍길동
한 줄 소개: 카페 인스타 SNS 운영 사장
SNS 채널: instagram.com/honggildong_cafe
힉스필드 풀코스 졸업: 2026-XX-XX
```

왜 표지 먼저? 포트폴리오 첫 인상이에요. 클라이언트나 고용주가 첫 페이지 보고 본인 누구인지 1초 안에 파악할 수 있게요.

**4단계 · 채널×모델 매트릭스 페이지 추가**
표지 페이지 아래에 새 하위 페이지 만드세요.
페이지 제목: "채널 × 모델 매트릭스 SSOT".
1단계에서 캡처한 매트릭스를 노션 표 형식으로 옮겨 적으세요.
노션은 표 만들기 쉬워요. "/table" 입력하면 표 자동 생성.

왜 매트릭스 페이지? 본인이 다음 콘텐츠 만들 때마다 이 페이지 열어서 1초 결정. SSOT의 진짜 역할이에요.

**5단계 · ACT 2 이미지 갤러리 페이지 추가 (5편)**
또 다른 하위 페이지 만들기.
페이지 제목: "ACT 2 · 이미지 5편 갤러리 (4~8강)".
5편 결과물 이미지 다 업로드.
각 이미지 옆에 만든 날짜·사용 모델·크레딧 사용량 기록.

왜 ACT별 분리? 22편 한 페이지에 다 올리면 페이지 무거워져요. ACT별로 분리하면 보기 편해요.

**6단계 · ACT 3 영상·ACT 4 Marketing Studio·ACT 5 실전 페이지 추가**
같은 흐름으로 ACT 3·4·5 페이지 4개 더 만들어주세요.
ACT 3 영상 5편(9~13강).
ACT 4 Marketing Studio 4편(14~17강).
ACT 5 실전·종합 8편+(18~19강 + 한 주 7편 캘린더).

각 페이지에 결과물 + 만든 날짜 + 사용 모델·모드 + 크레딧 기록.

왜 ACT별 페이지? 본인 학습 흐름이 ACT 1→5 순서로 나아간 거 한 눈에 보여요. 6개월 후 다시 봐도 흐름 명확.

**7단계 · 다음 단계 페이지 추가 (졸업 후 계획)**
마지막 하위 페이지 만들기.
페이지 제목: "졸업 후 다음 단계 (2026-XX 시작)".
콘셉트 8번에서 안내한 5가지 다음 단계 중 본인이 시도할 1~3가지 적기.

예:
- 매월 Plus 플랜 결제 유지.
- 본인 인스타 매주 한 주 캘린더 1세트 게시.
- 6개월 후 1,000 팔로워 목표.

왜 다음 단계? 졸업이 끝이 아니라 시작이에요. 다음 단계 적어두면 흐름 잃지 않아요.

**8단계 · 노션 페이지 공유 링크 만들기 (선택)**
표지 페이지 우측 상단 "Share" 또는 "공유" 버튼 클릭.
"Publish to Web" 또는 "웹에 게시" 옵션 활성화.
공유 링크가 생성돼요. 클라이언트·고용주에게 이 링크 보내면 노션 회원 아니어도 페이지 볼 수 있어요.

왜 공유? 본인 포트폴리오를 외부에 보여줄 수 있어요. 일자리·프리랜스 의뢰 받을 때 활용 가능.

**9단계 · 졸업 인증 사진 본인 SNS에 게시 (선택)**
노션 표지 페이지 스크린샷 + "힉스필드 풀코스 졸업 · 22+개 콘텐츠 만든 후기" 캡션으로 본인 인스타에 게시해보세요.
지금까지 만든 결과물 일부도 함께 게시하면 시청자에게 본인 학습 여정 공유.

왜 게시? 본인 학습 기록 + 채널 콘텐츠 + 시청자 신뢰 형성 동시에. 일석삼조예요.

**10단계 · 졸업 축하 + 다음 주 캘린더 미리 작성**
오늘이 졸업이지만 진짜 시작은 내일부터예요.
다음 주 캘린더 미리 작성해두세요. 19강에서 배운 한 주 7편 흐름 그대로.
1개월 후 본인 채널이 어떻게 변할지 기대돼요.

왜 다음 주? 졸업 후 흐름 끊기지 않게 즉시 다음 주 계획 세우는 게 핵심이에요.

졸업 축하해요. 셰프 30명을 다 만나본 본인은 이제 어떤 콘텐츠도 만들 수 있는 사람이에요.

> ❌ **실패 시**: 수업 채널에 캡처 + "20강 Step 1" 키워드로 질문

## 🐛 자주 발생하는 문제

> 💡 본 섹션은 회차 진행 후 학생 질문이 누적되면 갱신됩니다.

| 증상 | 원인 | 해결 |
|---|---|---|
| 명령어가 안 먹힘 | 오타 또는 폴더 위치 | `pwd`로 현재 위치 확인 후 `cd`로 이동 |
| 결과가 화면과 다름 | 직전 단계 누락 | 이전 Step 완료 여부 체크 |
| 한국어 안 나옴 | 프롬프트에 한국어 부재 | "한국어로 답해주세요" 추가 |
