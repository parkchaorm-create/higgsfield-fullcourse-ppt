## 1️⃣ Step 1 · 시연 1

이제 진짜 B 옷가게 셀러 페르소나로 시네마틱 인스타 릴스 광고 영상 1편 만들어볼게요.

**1단계 · 힉스필드 접속 + Cinema Studio 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 메뉴 중 "Cinema Studio" 클릭하세요. 글로벌 네비 10번째 메뉴예요.
또는 메인 화면 7개 액션 카드 중 "Cinema Studio 3.5" 카드 클릭해서 들어갈 수 있어요.

왜 Cinema Studio? 9~12강의 Video 메뉴와 다른 별도 메뉴예요. 카메라 동작 골라 쓰는 게 핵심 차이라 메뉴가 따로 있어요.

**2단계 · 작업 화면 둘러보기**
Cinema Studio 작업 화면 들어가면 일반 Video 메뉴와 좀 달라요.
왼쪽에 카메라 프리셋 메뉴 (Zoom·Movement·Angle·Rotate 4그룹)
가운데 프롬프트 입력 박스
오른쪽 또는 아래 영상 미리보기 영역
이런 구조예요.

왜 둘러보기? 카메라 프리셋 메뉴 위치를 먼저 알아두면 다음 단계가 쉬워요.

**3단계 · 한국어 프롬프트 입력 (인물·배경 묘사)**
가운데 프롬프트 박스에 인물·배경만 한국어로 묘사하세요. 카메라 동작은 빼고요.

```prompt
20대 한국 여성 모델이 미니멀 옷가게 매장에서 흰 셔츠와 베이지 와이드 팬츠를 입고 자연스럽게 걷는 모습, 따뜻한 자연광, 밝은 분위기, 8초
```

왜 카메라 동작 빼고? Cinema Studio에서는 카메라 동작을 메뉴에서 골라 클릭하기 때문이에요. 프롬프트에는 인물·옷·배경·조명만 적으면 돼요.

**4단계 · 카메라 프리셋 선택 (트래킹샷)**
왼쪽 카메라 프리셋 메뉴에서 "Movement" 그룹 클릭.
그 안에서 "Tracking Shot" 또는 "Follow"라고 되어있는 프리셋 클릭하세요.

왜 트래킹샷? 모델이 걷는 영상에는 트래킹샷(인물을 따라가는 시점)이 정답이에요. 옷의 움직임·실루엣을 가장 잘 보여줘요.

**5단계 · Aspect Ratio 9:16 + 길이 8초 선택**
종횡비 옵션에서 "9:16" 클릭하세요. 인스타 릴스 표준이에요.
영상 길이 옵션에서 "8s"(8초) 클릭하세요.

왜 8초? 인스타 릴스 광고는 6~10초가 황금 시간이에요. 8초면 모델이 걷는 동작 1.5번 정도 들어가서 옷 자연스러움 충분히 보여줘요.

**6단계 · Generate(만들기) 버튼 클릭**
"Generate" 클릭하세요. 크레딧 차감 메시지 미리 확인하고 누르세요.
Cinema Studio 영상은 약 20~40크레딧 정도예요. 12강 Veo Pro와 비슷한 수준이에요.
영상 만드는 데 3~5분 걸려요.

왜 미리 확인? 카메라 컨트롤 추가 옵션이 있어서 일반 영상보다 좀 더 들어요. 미리 확인해서 마음 편하게 시도하세요.

**7단계 · 시네마틱 결과 확인**
완료되면 화면에 8초 영상이 떠요. 재생 버튼 누르세요.
모델이 자연스럽게 걷고, 카메라가 부드럽게 따라가는지 확인하세요.
9강 Kling 첫 영상과 직접 비교해보세요. 카메라 동작이 얼마나 더 시네마틱한지 두 눈으로 느끼시면 돼요.

왜 비교 확인? Cinema Studio의 진짜 가치를 직접 체감하는 순간이에요. "왜 이 메뉴가 따로 있는지" 답이 나와요.

**8단계 · 다른 카메라 프리셋으로 1편 더 (선택)**
시간 여유 있으면 같은 프롬프트에 다른 카메라 프리셋 골라서 1편 더 만들어보세요.
"Angle" 그룹 → "Low Angle" 클릭 → Generate.
같은 모델·같은 옷이지만 카메라 각도만 바뀐 영상이 나와요.
로우앵글이 모델 키 더 커 보이고 옷 라인 더 길어 보이는 효과 직접 확인하세요.

왜 다른 프리셋? 70+ 프리셋의 효과를 직접 체험하는 거예요. 본인 콘텐츠에 어떤 프리셋이 맞는지 감 잡는 학습이에요.

**9단계 · Download + Assets 저장**
마음에 드는 영상 위에 마우스 올려서 "Download" + "Save to Assets" 둘 다 클릭하세요.
영상 파일(.mp4)로 컴퓨터에 저장돼요.

왜 두 곳? 컴퓨터 저장은 인스타 광고에 올릴 때 필요. Assets 저장은 14강 Marketing Studio에서 광고 영상 활용할 때 가져다 쓰려고요.

**10단계 · 폰으로 옮겨서 인스타 릴스 광고로 업로드**
다운로드한 영상 파일을 카카오톡 본인한테 보내거나 구글 드라이브로 폰에 옮기세요.
인스타 앱 → 새 게시물 → 릴스 → 영상 선택 → 9:16 그대로 → 음악·텍스트 추가는 선택 → 게시.
또는 인스타 광고 매니저(Meta Ads)에 직접 업로드 → 광고 캠페인 만들기.

왜 인스타 릴스 광고? B 옷가게 셀러님 메인 채널이에요. 시네마틱 카메라 동작이 들어간 영상이 일반 영상보다 광고 효과 1.5~2배 높다는 광고 데이터 있어요. 출처: 메타 광고 가이드.
