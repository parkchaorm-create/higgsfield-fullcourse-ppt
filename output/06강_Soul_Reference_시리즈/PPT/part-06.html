📝 강사 노트
B 옷가게 셀러님이 5강 끝나고 이런 생각 하셨을 수 있어요. "내 얼굴 학습은 좀 부담스럽고, 다른 방법은 없나요?" "인스타 릴스에 옷 시리즈 5컷 한 번에 올리고 싶은데 매번 분위기가 다르게 나와요." "참고로 보여주고 싶은 사진은 있는데, 글로 설명하기 어려워요." 이 고민들 오늘 한 방에 풀려요. 힉스필드에 Soul Reference(솔 레퍼런스)라는 기능이 있어요. 참고 사진 1장 셰프한테 보여주면서 "이 분위기로 5컷 더 만들어줘" 하면 끝이에요. 글로 설명할 필요 없어요. 사진 1장이 1,000 단어보다 정확해요. 얼굴 학습도 안 하고요. 그냥 분위기 복사예요. [CONCEPT 1] Soul Reference(솔 레퍼런스 · 이미지 → 이미지 변환)부터 짚을게요. Soul Reference는요. 셰프한테 사진 1장 보여주면서 "이 분위기로 비슷하게" 부탁하는 기능이에요. 친구한테 식당 추천받을 때 비슷해요. "어떤 분위기 식당이야?" 물어보면 친구가 사진 1장 보여주잖아요. "이런 분위기" 한 마디면 끝. Soul Reference도 똑같아요. 우리가 셰프한테 사진 1장 보여주면 셰프가 "아, 이 분위기 원하시는 거군요" 하고 그대로 만들어줘요. 글로 "따뜻한 갈색 톤, 자연광, 미니멀 인테리어, 옷걸이에 걸린 옷..." 이렇게 길게 설명할 필요가 없어요. 사진 1장이 1,000 단어보다 정확해요. [CONCEPT 2] 5강 Soul ID와 6강 Soul Reference의 차이를 명확히 짚을게요. Soul ID는 "내 얼굴" 학습이에요. 셰프 앞치마에 내 사진 5장 새겨두고, 매번 같은 얼굴로 만들어요. Soul Reference는 "분위기 복사"예요. 사진 1장 보여주면서 "이 분위기로" 부탁하는 거예요. 얼굴 학습 안 해요. B 옷가게 셀러님 같은 경우는요. 내 얼굴 보여주는 채널 아니잖아요. 옷 자체가 주인공이에요. 그래서 Soul ID보다 Soul Reference가 딱 맞아요. "이 분위기 옷걸이 사진처럼" 또는 "이 모델 자세처럼" — 분위기 복사로 시리즈 만들기 좋아요. 얼굴 학습 5~10분 기다릴 필요도 없고요. [CONCEPT 3] 참고 사진(Reference · 참고 이미지 첨부) 잘 고르는 법을 알려드릴게요. 참고 사진 1장이 결과 80% 결정해요. 잘 고르는 게 핵심이에요. 3가지 원칙이에요. 첫째, 깨끗한 분위기. 잡다한 배경 없는 사진. 옷이나 모델이 주인공인 사진. 둘째, 비슷한 채널 분위기. 본인 인스타 분위기랑 닮은 사진. 갑자기 너무 화려한 사진 가져오면 셰프가 헷갈려요. 셋째, 핀터레스트(Pinterest · 이미지 검색 사이트)나 인스타에서 1장 골라오기. 본인 마음에 진짜 쏙 드는 1장이면 결과도 만족스러워요. "옷걸이 미니멀 인테리어"·"자연광 편집숍 분위기" 이런 키워드로 검색하면 좋은 참고 사진 많이 나와요. [CONCEPT 4] 9:16 세로 액자(인스타 릴스용)를 짚을게요. 4강에서 1:1 정사각 액자(인스타 피드용) 한 번 다뤘어요. 오늘은 9:16 세로 액자예요. 핸드폰 화면 통째로 차지하는 모양이요. 인스타 릴스·틱톡·유튜브 쇼츠가 다 9:16이에요. B 옷가게 셀러님은 인스타 릴스가 메인 채널이라고 했죠. 그래서 9:16이 맞아요. 1:1로 만들면 릴스에 올릴 때 위아래 잘려요. 처음부터 9:16으로 만들어야 안전해요. 종횡비(Aspect Ratio) 선택만 1:1에서 9:16으로 바꾸면 끝이에요. 다른 건 똑같아요. [CONCEPT 5] Soul Location(솔 로케이션 · 시리즈용 배경)을 살짝 짚어둘게요. 오늘 핵심은 Soul Reference인데요. 함께 자주 쓰는 Soul Location도 알아두면 좋아요. Soul Location은요. 시리즈 5컷 다 같은 배경에서 찍는 것처럼 만들어주는 기능이에요. "카페 인테리어"·"옷가게 매대" 같은 배경 1개 정해두면 5컷 다 그 배경에서 나와요. B 옷가게 셀러님이 매장 분위기 일관되게 보여주고 싶을 때 쓰면 좋아요. 오늘은 Soul Reference만 깊게 하고요. Soul Location은 다음 시리즈 작업 때 한 번 더 시도해보세요.

📝 강사 노트
오늘 한 일 정리할게요. 첫째. Soul Reference(솔 레퍼런스)는 사진 1장 보여주면서 "이 분위기로" 부탁하는 기능이에요. 글로 설명 안 하고 사진 1장으로 분위기 복사하는 거예요. 5강 Soul ID는 얼굴 학습, 6강 Soul Reference는 분위기 복사 — 이렇게 구분하세요. 둘째. B 옷가게 셀러 페르소나로 참고 사진 1장 + 한국어 프롬프트 5종 + 9:16 세로 액자 = 일관 분위기 옷 시리즈 5컷 받았어요. 결과물 누적 3번 ✓ (4강 1장 + 5강 2장 + 6강 5장 = 8장). 셋째. 참고 사진은 깨끗하고 본인 채널 분위기랑 닮은 1장으로 고르는 게 결과 80%를 결정해요. 핀터레스트 활용해서 본인 마음에 쏙 드는 1장 찾는 게 핵심이에요.
