## 1️⃣ Step 1 · 시연 1

이제 진짜 C 식당 사장 페르소나로 4K 음식 사진 2~3장 만들어볼게요.

**1단계 · 힉스필드 접속 + Image 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 두 번째 메뉴 "Image" 클릭하세요.

왜 Image? 오늘은 음식 사진(이미지) 만들거예요. Video는 영상용이에요.

**2단계 · Nano Banana Pro 카드 선택**
모델 카드들 중에서 "Nano Banana Pro" 카드를 찾으세요.
카드 위에 노란 "UNLIMITED" 배지가 붙어있어요. 그게 표시예요.
클릭하세요.

왜 Nano Banana Pro? 4K 음식 사진 전문 셰프예요. Soul 2.0(인물 사진 셰프)이 아니라 이 셰프예요.

**3단계 · UNLIMITED 배지 위치 직접 확인**
Nano Banana Pro 작업 화면 들어가면 화면 어딘가에 "UNLIMITED" 또는 "Free for Plus subscribers" 같은 안내가 보여요.
크레딧 차감 옆 표시도 "0 credits" 또는 "Unlimited"로 바뀌어 있을 거예요.

왜 직접 확인? "이 셰프는 정말 무제한이구나" 두 눈으로 확인하면 마음 편하게 시도할 수 있어요. 가짜 무제한이면 안 되잖아요.

**4단계 · 한국어 프롬프트 입력 (16:9 가로 음식 사진)**
화면 가운데 텍스트 입력 박스에 한국어로 그대로 입력하세요.

```prompt
한국식 비빔밥, 흰 도자기 그릇에 담긴 모습, 나무 식탁 위, 따뜻한 자연광, 위에서 내려다본 구도, 4K 고화질
```

왜 이 프롬프트? "비빔밥"으로 음식 + "흰 도자기 그릇"으로 그릇 + "나무 식탁"으로 배경 + "따뜻한 자연광"으로 조명 + "위에서 내려다본 구도"로 음식 사진 표준 구도 + "4K 고화질"로 화질까지 한 번에 지시했어요.

**5단계 · Aspect Ratio 16:9 선택 + Generate**
종횡비 옵션에서 "16:9"를 클릭하세요.
"Generate" 버튼 클릭하세요.
20~40초 기다리면 4K 비빔밥 사진이 화면에 떠요.

왜 16:9? 페북 일반 게시물에 가장 많이 보이는 모양이에요. 첫 컷은 페북용으로 받아요.

**6단계 · 결과 확인 + Download**
사진이 마음에 들면 사진 위에 마우스 올려서 "Download" 클릭하세요.
컴퓨터에 저장돼요.
크레딧 잔액 보세요. 0 차감됐을 거예요. UNLIMITED 매력 직접 체감하는 순간이에요.

왜 크레딧 확인? "정말 무제한이구나" 직접 확인하면 다음 시도도 부담 없이 할 수 있어요.

**7단계 · 같은 메뉴를 1:1 정사각으로 다시**
프롬프트는 그대로 두고 종횡비만 "1:1"로 바꿔주세요.
"Generate" 다시 클릭.
이번엔 정사각 액자로 같은 비빔밥 사진이 나와요.

왜 1:1? 페북 광고용·인스타 피드 겸용으로 정사각 한 장 더 만들어둬요. 한 번 만들 때 두 액자 다 받으면 활용도 2배예요.

**8단계 · 다른 메뉴 1~2장 더 (선택)**
시간 여유 있으면 다른 메뉴 1~2장 더 만들어보세요. 무제한이니까 부담 없어요.
프롬프트만 음식 이름 바꿔주면 돼요.

```prompt
한국식 불고기, 검은 무쇠 팬에 담긴 모습, 나무 식탁 위, 따뜻한 자연광, 45도 각도 구도, 4K 고화질
```

```prompt
한국식 잡채, 흰 도자기 접시에 담긴 모습, 나무 식탁 위, 따뜻한 자연광, 위에서 내려다본 구도, 4K 고화질
```

왜 여러 메뉴? 식당 메뉴 사진은 보통 5~10가지 정도 필요해요. 무제한 셰프 한 번 만난 김에 여러 장 만들어두면 한 달치 콘텐츠가 한 번에 끝나요.

**9단계 · 모두 Assets로 저장**
만든 사진들 다 "Save to Assets" 또는 "Download" 눌러서 저장하세요.
힉스필드 안 Assets 보관함에 저장해두면 나중에 페북·인스타·카카오톡 어디든 가져다 쓸 수 있어요.

왜 두 곳 저장? 컴퓨터 다운로드는 폰으로 옮길 때 필요. Assets 보관은 다른 회차에서 영상 만들 때 가져다 쓸 때 필요해요.

**10단계 · 페북·카카오에 업로드**
다운로드한 사진들 카카오톡 본인한테 보내거나 구글 드라이브로 폰에 옮기세요.
페북 앱 → 새 게시물 → 사진 선택 → 16:9 가로 사진 그대로 업로드.
카카오톡 채널·비즈니스 채널 → 새 메시지 → 사진 선택 → 1:1 정사각 활용.

왜 채널별 매칭? 같은 4K 음식 사진이라도 페북 일반 게시물엔 16:9, 카카오톡 광고엔 1:1이 더 잘 어울려요. 액자만 골라 쓰면 한 번 만들고 채널마다 활용해요.

| 증상 | 원인 | 해결 |
|---|---|---|
| 명령어가 안 먹힘 | 오타 또는 폴더 위치 | `pwd`로 현재 위치 확인 후 `cd`로 이동 |
