# 5강 실습 · 클로드 코드 풀코스

> **소요 시간**: 25~35분 (수업 시간 내 완결)
> **목표 산출물**: 클로드 코드 풀코스 직접 따라하기
> **안전망**: 1개 단계 중 마지막 직전까지 도달하면 합격
> **선행 조건**: 이전 회차 완료 (4강 산출물)

## 🚀 Step 0 · 빠른 셋업 (3분)

1. 책상 위 음료수 한 잔 준비
2. 휴대폰 무음 모드
3. 다른 창 모두 닫기 (집중)
4. 터미널(또는 PowerShell) 열기
5. 본 회차 PPT 강사 노트 화면 함께 띄워두기

## 1️⃣ Step 1 · 시연 1

이제 D 트레이너 페르소나로 Soul ID 학습부터 첫 결과물까지 만들어볼게요. 본인 얼굴로 진행하면 더 좋아요.

**1단계 · 본인 사진 5장 준비**
폰 사진첩에서 본인 얼굴 사진 5장 골라서 컴퓨터로 옮겨주세요.
구체적으로 이런 5장 추천해요. 정면 1장·옆얼굴 1장·살짝 위에서 1장·미소 1장·진지한 얼굴 1장.
모자·선글라스 안 쓴 사진으로요.

왜 5장? 너무 적으면 셰프가 얼굴 못 외워요. 너무 많으면 학습 시간 길어져요. 5장이 균형점이에요.

**2단계 · 힉스필드 접속 + Character 메뉴 클릭**
크롬 브라우저로 다음 주소 들어가주세요.

```paste
https://higgsfield.ai
```

화면 위쪽 메뉴 중 "Character"(8번째 메뉴)를 찾아서 클릭하세요.
Character는 한국어로 "캐릭터(인물 일관성)"예요.

왜 Character? Soul ID는 Character 메뉴 안에 있어요. Image 메뉴 아니에요.

**3단계 · Soul ID 또는 "Create New Character" 클릭**
Character 페이지에 들어가면 "Soul ID" 또는 "Create New Character"(새 캐릭터 만들기) 버튼이 보여요.
클릭하세요.
새 캐릭터 만들기 화면이 열려요.

왜 이 버튼? 셰프 앞치마에 내 이름 새기기 시작하는 시작점이에요.

**4단계 · 캐릭터 이름 입력**
입력 박스에 캐릭터 이름 입력하세요. 한국어로 그대로 입력해도 돼요.

```paste
D트레이너 (또는 본인이름_트레이너)
```

왜 이름? 나중에 셰프가 "어떤 캐릭터 만들어드릴까요?" 물어볼 때 이 이름으로 골라요. 본인 알아보기 쉬운 이름으로.

**5단계 · 사진 5장 업로드**
"Upload Photos" 또는 사진 업로드 박스가 보여요.
1단계에서 준비한 사진 5장을 드래그해서 박스 안에 끌어다 놓거나, "Browse" 클릭해서 선택하세요.
업로드 진행 바가 끝까지 차면 완료예요.

왜 5장 한 번에? 셰프가 다양한 각도·표정 한 번에 봐야 얼굴을 입체적으로 외워요.

**6단계 · "Train" 또는 "Create" 버튼 클릭**
사진 업로드 끝나면 화면 아래 "Train Soul ID" 또는 "Create Character" 버튼이 활성화돼요.
클릭하세요.
"학습 시작" 메시지가 떠요.

왜 이 버튼? 셰프가 진짜 앞치마에 내 사진 새기기 시작하는 순간이에요. 5~10분 기다리는 시간 시작이에요.

**7단계 · 학습 진행 화면 확인 + 5~10분 대기**
"Training in progress..." 같은 메시지가 떠요. 진행 바가 천천히 차요.
이 시간 동안 다른 일 하셔도 돼요. 화면 닫아도 학습은 백그라운드에서 계속 진행돼요.

왜 기다림? 셰프가 사진 5장 분석해서 본인 얼굴 입체 모델 만드는 중이에요. 한 번 학습하면 평생 써먹어요.

**8단계 · 학습 완료 알림 확인**
5~10분 지나면 화면 우측 상단 알림(Notification · 벨 아이콘)에 빨간 배지가 떠요.
클릭하면 "Your character is ready!" 메시지가 보여요.
또는 Character 페이지 다시 들어가면 본인 캐릭터 카드가 "Ready" 상태로 보여요.

왜 알림? 다른 일 하다가 알림으로 완료 시점 알 수 있어요. 화면 앞 계속 안 있어도 돼요.

**9단계 · Image 메뉴로 이동 + Soul 2.0 선택 + 캐릭터 적용**
이제 본인 얼굴로 사진 만들어볼게요.
화면 위쪽 "Image" 메뉴 클릭하세요.
"Soul 2.0" 선택하세요.
작업 화면에 들어가면 "Use Character" 또는 "Apply Soul ID" 옵션이 보여요. 클릭해서 8단계에서 만든 캐릭터(D트레이너)를 선택하세요.

왜 캐릭터 적용? 이 단계가 셰프한테 "오늘은 이 사람 얼굴로 만들어줘" 외치는 신호예요.

**10단계 · 운동 시연 프롬프트 입력 + 1:1 종횡비 + Generate**
캐릭터 적용된 상태에서 한국어로 프롬프트 입력하세요.

```prompt
헬스장에서 덤벨을 들고 운동하는 30대 남성 트레이너, 밝은 자연광, 정면 구도, 미소 짓는 모습
```

종횡비 "1:1"(인스타 피드용) 선택하고 "Generate" 클릭하세요.
20~40초 후 본인 얼굴로 운동 시연하는 사진이 떠요.

왜 운동 시연? D 트레이너 페르소나에 맞는 콘텐츠예요. 인스타 시리즈 첫 컷 만드는 거예요.

**11단계 · 같은 캐릭터로 1장 더 만들기 (시리즈 확인)**
프롬프트만 바꿔서 다시 Generate 하세요.

```prompt
헬스장 거울 앞에서 스트레칭하는 30대 남성 트레이너, 자연광, 측면 구도
```

새 사진이 떠요. 1번째 사진과 비교해보세요. 자세는 다른데 얼굴은 같아야 해요.
이게 일관 캐릭터의 마법이에요.

왜 두 번째? 시리즈 콘텐츠 가능성을 직접 눈으로 확인하려고요. "같은 얼굴 다른 자세" — 이게 진짜 시리즈 콘텐츠 시작점이에요.

> ✅ **성공 신호**: 위 마지막 결과 화면이 그대로 나옴

> ❌ **실패 시**: 수업 채널에 캡처 + "5강 Step 1" 키워드로 질문

## 🐛 자주 발생하는 문제

> 💡 본 섹션은 회차 진행 후 학생 질문이 누적되면 갱신됩니다.

| 증상 | 원인 | 해결 |
|---|---|---|
| 명령어가 안 먹힘 | 오타 또는 폴더 위치 | `pwd`로 현재 위치 확인 후 `cd`로 이동 |
| 결과가 화면과 다름 | 직전 단계 누락 | 이전 Step 완료 여부 체크 |
| 한국어 안 나옴 | 프롬프트에 한국어 부재 | "한국어로 답해주세요" 추가 |
