📝 강사 노트
B 옷가게 셀러님이 평소 인스타 운영하면서 이런 고민 하셨을 거예요. "옷 사진은 잘 찍었는데 모델이 없어서 옷만 사진으로 보여주니 임팩트가 약해요." "모델 한 분 부르면 하루 촬영비 30~50만원이고 옷마다 부르기 부담돼요." "같은 옷을 여러 모델·여러 자세로 찍어서 시리즈 만들고 싶은데, 이게 가능해요?" 이 고민, 오늘 한 방에 풀려요. 힉스필드 Marketing Studio에 Pro Virtual Try-On(프로 가상 착용) 모드가 있어요. "종이 인형 옷 갈아입히기" 비유가 정확해요. 어렸을 때 종이 인형에 옷 종이 갖다 대면 입은 모습 되었던 거 기억나시죠. 디지털로 그 마법이 가능해진 거예요. 옷 사진 1장 + 모델 사진 1장 = 입은 모습 즉시 생성. 모델은 본인 사진도 OK, AI 가상 모델도 OK예요. 가상 모델은 다양한 인종·체형·자세로 무한 변형 가능해요. 오늘 같은 옷 한 벌을 5컷 다른 모델로 변형해서 인스타 릴스 정지샷 시리즈 만들어봐요. [CONCEPT 1] Pro Virtual Try-On(프로 가상 착용 · 옷·뷰티 가상 입히기) 부터 짚을게요. "가상 착용"은요. 글자 그대로 "가상으로 옷을 입어보는 것"이라는 뜻이에요. 실제 모델 부르지 않고 사진만으로 입은 모습을 보여주는 기술이에요. 온라인 쇼핑몰 가본 적 있으시죠. 요즘 일부 쇼핑몰에 "AR로 입어보기" 기능 있는 거 보셨을 거예요. 그게 바로 가상 착용 기술이에요. 힉스필드 Pro Virtual Try-On은요. 의류 셀러용으로 특화된 가상 착용 기능이에요. 종이 인형 옷 갈아입히기 비유가 정확해요. 종이 모델에 옷 종이 갖다 대면 입은 모습 즉시 보였잖아요. 그걸 디지털로 옮긴 거예요. 9가지 Marketing Studio 모드 중 8번째 모드예요. 14강 Product Review·15강 UGC와는 다른 카테고리(의류·뷰티 셀러용)예요. B 옷가게 셀러님처럼 의류 판매하시는 분들이 가장 많이 쓰는 마법 같은 기능이에요. [CONCEPT 2] 옷 사진 + 모델 사진 업로드 흐름을 짚을게요. Pro Virtual Try-On 모드에 진입하면 두 가지 업로드 박스가 떠요. 첫째 박스, "Garment(가먼트 · 옷 사진)" 업로드. 둘째 박스, "Model(모델 사진)" 업로드. 옷 사진은요. 본인이 판매하는 옷을 평평하게 펼쳐 찍은 사진이 가장 좋아요. 또는 마네킹에 입혀진 사진도 OK예요. 배경이 깔끔할수록 결과 정확해요. 모델 사진은 두 가지 옵션이 있어요. 첫째, 본인이나 지인 사진 업로드(실제 모델 입은 모습 받기). 둘째, AI 가상 모델 선택(다양한 인종·체형·자세에서 골라 입히기). B 옷가게 셀러님 같은 분은 가상 모델이 더 활용도 높아요. 모델 한 분이 못 보여주는 다양한 체형·인종·연령대를 한 번에 보여줄 수 있거든요. [CONCEPT 3] 입은 모습 즉시 생성을 짚을게요. 업로드 마치고 Generate(만들기) 버튼 누르면 약 1~2분 안에 입은 모습이 나와요. 14강 Marketing Studio(5~10분)·15강 UGC(5~10분)보다 빨라요. Pro Virtual Try-On은 옷+모델 결합만 하기 때문에 가벼운 작업이에요. 크레딧 비용도 30~50크레딧으로 14강·15강(50~100)보다 절반 정도예요. 옷의 색상·질감·로고가 그대로 유지되는지 확인하세요. 가끔 옷이 살짝 변형되는 경우가 있어요. 그럴 땐 다시 생성하거나 옷 사진을 더 깔끔한 걸로 교체하면 돼요. 입은 모습이 어색하지 않은지(예: 옷이 모델 몸에 안 맞는 비율) 확인하세요. 어색하면 모델 사진을 다른 체형으로 바꿔서 재시도해요. [CONCEPT 4] 다양한 모델로 5컷 변형을 짚을게요. 같은 옷 + 5명 다른 가상 모델 = 5컷 시리즈 만들 수 있어요. 각 모델은 인종·체형·자세 다 다르게 선택 가능해요. 첫 번째 컷, 20대 한국 여성 모델 + 자연스러운 자세. 두 번째 컷, 30대 한국 여성 모델 + 옆모습. 세 번째 컷, 20대 동남아 여성 모델 + 워킹 자세. 네 번째 컷, 30대 서양 여성 모델 + 정면 자세. 다섯 번째 컷, 40대 한국 여성 모델 + 자연스러운 미소. 이렇게 5컷 만들면 한 옷에 5가지 다른 분위기가 나와요. 인스타 릴스 정지샷 시리즈로 활용할 때 시청자가 다양한 체형에서 옷이 어떻게 보이는지 한 눈에 비교할 수 있어요. 같은 옷을 5컷 만들 때 크레딧은 약 150~250(컷당 30~50)이에요. [CONCEPT 5] 인스타 릴스 정지샷 변환을 짚을게요. Pro Virtual Try-On 결과는 정지 이미지(JPG·PNG)예요. 영상이 아니에요. 그래서 인스타 릴스로 활용하려면 정지샷 5컷을 영상 편집 앱에서 슬라이드쇼로 묶어야 해요. 6강 Soul Reference 정지샷을 릴스로 변환했던 거 기억나시죠. 간단한 방법은요. 인스타 앱 안에 "릴스" 메뉴에서 사진 5컷 선택 → 자동 슬라이드쇼 → 음악 추가. 이게 가장 쉬워요. 조금 더 정교하게 하려면 캡컷(CapCut)·인쇼트(InShot) 같은 무료 영상 편집 앱에서 슬라이드 전환 효과 추가하시면 돼요. 영상 길이는 15~20초가 적당해요. 너무 길면 시청 완료율 떨어져요.
